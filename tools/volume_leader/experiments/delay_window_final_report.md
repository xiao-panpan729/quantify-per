# 延迟窗口实验 — 最终报告（2026-05-27）

## 实验目的

确定 ★买 信号出现后，MA5>MA10>MA20 条件在多少根K线内跟上，仍能保持可接受的胜率和收益。

## 实验环境

| 参数 | 值 |
|------|-----|
| 标的 | 47只量领强者（volume_leader_universe.json） |
| 周期 | min5 |
| 回测范围 | 6个月 |
| 入场模式 | MA级：★买 + MA5>MA10>MA20 + 无死叉(20根回溯) + 60分EXPMA黄线上方 + 日线强势/二级 |
| 入场价 | (low + close) / 2 |
| 止损 | 开仓bar的波段最低点（向前找最近死叉，最远40根，取区间最低low） |
| 出场模式（新） | 止损 > 减仓卖 |
| 减仓卖出条件 | 死叉 或 (★卖 + close<MA5 + 无金叉(20根回溯) + 15分EXPMA黄线下方) |
| 出场价（新） | close |
| MFE/MAE | 入场到出场区间最高high/最低low vs 入场价 |

### 新旧差异分析

| 窗口 | 胜率差(新-旧) | 均收益差(新-旧) | 原因 |
|------|-------------|----------------|------|
| 0 | -6.4pt | -0.25pt | 新出场(★卖+close<MA5)比旧(仅死叉)更早触发，截断尾部利润 |

**解读**：新出场逻辑在高质量信号（N=0同框）中影响最大（-6.4pt胜率），因为上涨趋势中★卖+close<MA5 比死叉更早出现，提前离场导致少赚尾部利润。在低质量信号（N=5延迟）中影响很小，因为信号本身就差。

## 结论

1. **N=1 是甜点**：延迟1根K线，笔数从47翻到117（+149%），胜率从72.3%降到67.5%（-4.8pt），均收益从+6.60%降到+4.87%。这是最后一个胜率>65%的窗口。

2. **N=2 是断崖**：胜率从67.5%暴跌到56.0%（-11.5pt），均收益降到+3.64%。无论用什么出场逻辑，N=2的断崖始终存在。

3. **通达信公式参数**：买侧 COUNT(星买,2)=本根+后1根=延迟1根窗口，卖侧 COUNT(星卖,3)=本根+后2根=延迟2根窗口，与实验结论一致。

4. **MA级 > 金叉级**：MA级笔数多2.5倍，胜率高3.9pt，均收益高3.33pt。金叉级的EXPMA金叉条件过滤了部分盈利交易。

5. **新出场逻辑更保守**：减仓卖（★卖+close<MA5+safe/死叉）比旧逻辑（仅死叉）更早离场，N=0时胜率低6.4pt但换来了更及时的下跌保护。
