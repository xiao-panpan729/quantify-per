# 动量评分系统 V3 — 通达信公式源码

**指标名称:** RSI势能2 (自定义指标)
**数据周期:** 日线
**数据量:** 1000根
**适用场景:** 板块指数RPS排名 / 个股势能评分

## 完整公式代码

```tdx
{************* 动量评分系统V3（全周期框架+短期增强）*************}

超短周期 := IF(VOL/MA(VOL,5)>1.5,3,5);  
超短强度 := ATAN((MA(C,超短周期)/REF(MA(C,超短周期),1)-1)*100)*57.3;

短期强度 := EMA( 
           ATAN((MA(C,13)/REF(MA(C,13),1)-1)*100)*57.3, 
           2);    


中期强度 := EMA(ATAN((MA(C,20)/REF(MA(C,20),1)-1)*100)*57.3,3);  
长期强度 := REF(ATAN((MA(C,60)/REF(MA(C,60),20)-1)*100)*57.3,0); 


量能因子 := MAX(VOL/REF(VOL,1), MA(VOL,5)/REF(MA(VOL,5),1)); 
强度放大系数 := IF(量能因子>1.8,1.25, IF(量能因子>1.3,1.15,1));


势能评分 := (
   超短强度*0.45*强度放大系数 + 
   短期强度*0.3 +             
   中期强度*0.2*0.85 +         
   长期强度*0.05              
)*100/60; 


连续强势天数 := BARSLAST(势能评分<REF(势能评分,1)); 
X_1 :(势能评分 + 连续强势天数*0.015)*0.1; 

{▉ 输出完整的四维度评分 ▉}
STICKLINE(X_1>0,0,X_1,2,0),COLORRED;
STICKLINE(X_1<0,0,X_1,2,0),COLORGREEN;
```

## Python 翻译说明

| TDX 函数 | Python 等效 |
|----------|-------------|
| `MA(C, N)` | `SMA(close, N)` |
| `REF(X, N)` | `X[-N-1]` (0-based) |
| `ATAN(x)*57.3` | `np.degrees(np.arctan(x))` |
| `EMA(X, N)` | 递归 `alpha=2/(N+1)` |
| `BARSLAST(条件)` | 条件为True到现在的Bar数 |
| `VOL` | `volume[-1]` |

**关键因子:** 日线价格系数 `*0.01`，所有运算在原始价格上进行。
