## 更新机制

1. `gen_daily_brief.py` 运行 → 从聚合消息（华尔街见闻/东方财富/财联社/金十）自动提取外资机构观点
2. 按板块分类 → 写入 `foreign_views/daily/YYYY-MM-DD.md` 当日快照
3. 同时增量追加到本文件对应板块表格（自动去重）
4. 聚合消息内容详细直接用；仅是标题的由 Claude Code 联网验证步骤（第0.3步）搜索补全

> 方向标识：↑ 增配/看多 | ↓ 减配/看空 | → 维持 | 🔄 评级调整
