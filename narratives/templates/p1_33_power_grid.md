# 电网/电力系统（特高压/智能电网/虚拟电厂）— P1 #33

## 当前核心逻辑（主线）

**AI数据中心+新能源并网双重驱动电网投资进入"超级周期"。** 新能源装机的快速增长（风光装机2026年预计突破2000GW）需要配套电网送出/消纳；AIDC（单数据中心300MW+）成为新型高密度负荷，对城市配电网的容量和可靠性提出全新挑战（见#6/#11）。

特高压是解决"三北"清洁能源→中东部负荷中心跨区输送的核心手段——2025年国网特高压投资超1000亿元，2026-2027年有望维持高位（"十四五"结转+"十五五"新增）。配电网的智能化改造（一二次融合设备/智能电表/故障定位）是长期持续的资本开支。

虚拟电厂（VPP）是新的商业模式——海量分布式光伏+储能+充电桩聚合参与电力市场交易，2025年试点范围扩大，虚拟电厂聚合商（恒实科技/国能日新等）进入商业化运营阶段。

## 旧逻辑（正在弱化）

电网投资是"缓慢而稳定"的——国网/南网每年电网投资~5000-6000亿，CAGR~5%，没有弹性。电网设备股给15-20x PE，没有估值溢价。虚拟电厂是"概念"——没有市场化交易机制，聚合商赚不到钱。

现在：AI数据中心成为新的高密度负荷，电网接入容量不足成为AIDC扩张的瓶颈。虚拟电厂的商业模式跑通了——政策明确支持+电力现货市场建设加快+分布式光伏配储普及。配电网的智能化投资增速从5%提升到15%+。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 特高压→跨区清洁能源输送 | 当前主线 | ★★★ | 2025年投资超1000亿，2026-2027年维持高位 |
| AIDC接入→配电网升级 | 并行 | ★★★ | AIDC是新型大负荷，配电网扩容/改造是刚需 |
| 配电网智能化→一二次融合设备 | 持续 | ★★★ | 智能电表/故障指示器/自动化开关持续放量 |
| 虚拟电厂→商业化运营 | 并行 | ★★☆ | 电力现货市场加快，VPP聚合商盈利模式落地 |
| 电力IT→电力市场交易系统 | 早期 | ★★☆ | 电力市场化改革催生交易/结算/预测软件需求 |
| 柔直输电→新型输电技术 | 早期 | ★★☆ | 柔性直流是深远海风电送出+大电网互联的关键技术 |

## 传导链

新能源装机↑ + AIDC负荷↑ → 电网接入和送出能力不足 → 国网/南网加大投资 → 特高压+配电网双线投资扩张 → 电网设备（变压器/开关/线缆/电表）需求爆发 → 电力IT/VPP等服务型环节受益

## 周期定位

**电网投资处于从1→10的加速期（AI+新能源双重驱动）。** 不同于新能源（光伏/储能）的产能过剩格局，电网是需求驱动而非供给驱动——不存在产能过剩问题，企业盈利能力稳定。
