# 锂资源（锂矿/盐湖提锂）— P1 #32

## 当前核心逻辑（主线）

**锂价在2023-2025年经历暴跌（60万→7万/吨）后进入底部区间，供需再平衡正在发生。** 高成本矿山（澳大利亚Liontown/中国江西云母矿）开始减产/停产，供给端出清信号出现。需求端仍维持15-20%增长（动力电池+储能+AIDC备电三驱动），但已不再是翻倍增速。

盐湖提锂成本最低（3-4万/吨），在当前锂价下仍有利润盈利，是未来锂供给的主要增量来源（南美盐湖+中国青海/西藏盐湖）。锂矿（锂辉石）成本在6-8万/吨，当前价格已接近成本线。

## 旧逻辑（正在弱化）

"锂是白色石油"——需求CAGR 30%+，供给跟不上，锂价会一直涨。2021-2022年的锂价暴涨（4万→60万/吨）使市场认为锂是"超级周期"，所有锂矿公司都值得投资。

现在锂从"稀缺品"变为"周期品"——全球锂资源足够丰富（南美盐湖+澳洲锂矿+非洲锂矿供给充分释放），锂价从60万暴跌至7万证明了供应的价格弹性。投资逻辑从"买锂价上涨"变为**"买低成本资源在底部出清后的价格修复"**。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 锂价底部→供给出清→价格修复 | 当前主线 | ★★★ | 高成本矿山减产，供给过剩最严重的时候已过 |
| 盐湖提锂→低成本+环保优势 | 并行 | ★★★ | 盐湖成本3-4万/吨，是未来锂供给的主力增量 |
| 海外锂资源→地缘风险溢价 | 并行 | ★★☆ | 南美/非洲锂矿出口风险增加，中国锂资源自给率需提升 |
| 锂回收→城市矿山 | 早期 | ★☆☆ | 电池退役潮到来后，回收锂将补充一部分供给 |
| 固态电池影响→远期需求不确定性 | 早期 | ★★☆ | 固态电池可能降低单位用锂量，影响远期需求预期 |

## 传导链

锂价跌至成本线→高成本产能出清→供需边际改善→锂价企稳→低成本资源（盐湖）扩产→需求（电池）增长15-20%→锂价温和回升→锂资源企业盈利修复

## 周期定位

**处于从1→10的周期底部出清期向"修复期"过渡。** 类比2015-2016年的碳酸锂——当时从4万涨到18万，驱动了一轮锂矿大行情。当前供给侧出清信号出现，需求端维持稳定增长，锂价修复的条件在积累。

## 明星标的

- 盐湖提锂：盐湖股份 / 藏格矿业 / 蓝科锂业 / 西藏矿业
- 锂矿：赣锋锂业 / 天齐锂业 / 中矿资源 / 永兴材料
- 锂云母：永兴材料 / 江特电机（成本较高，承压中）
- 海外锂矿：天齐锂业（SQM/Greenbushes持股）
