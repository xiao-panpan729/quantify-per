# 新能源整车/车身 — P0 #18

## 当前核心逻辑（主线）

**中国新能源车渗透率迈过50%分水岭，进入"存量竞争+全球化"新阶段。** 2025年中国新能源车渗透率突破50%，销量约1200万辆，增速从翻倍期进入20-30%的稳态增长期。行业主线从"电动化替代"切换为"智能化竞争"（见#17），价格战趋于理性——头部车企毛利率改善，尾部加速出清。

全球化是第二曲线——中国新能源车出口2024年突破200万辆，东南亚/欧洲/中东是主要市场。欧盟关税加征短期压制，但中国车企通过本地建厂（比亚迪匈牙利/奇瑞西班牙）绕过壁垒。

## 旧逻辑（正在弱化）

"电动化替代燃油车"是2020-2024年的核心叙事——新能源车渗透率从5%→50%，行业增速>50% CAGR，市场给的是"颠覆者估值"（PSx甚至零利润估值）。所有新势力都在烧钱抢份额，市场容忍亏损。

现在渗透率>50%后，增量从"替代"变为"升级"。市场估值体系从"颠覆者PS"切换为"成熟车企PE（15-20x）"。价格战从"烧钱抢份额"变为"保利润生存"。有规模效应+品牌溢价的头部车企胜出，没有差异化的小众品牌加速出清。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 智能化竞争→第二波差异化 | 当前主线 | ★★★ | 智驾+座舱是消费者决策核心，见#17/#19 |
| 全球化→中国车企出海 | 并行 | ★★★ | 海外毛利率更高，但关税/地缘风险增加 |
| 价格战趋缓→利润修复 | 持续 | ★★☆ | 尾部玩家出清，BOM成本下降推动毛利率改善 |
| 小米/华为→新玩家扰动格局 | 并行 | ★★★ | 小米SU7/华为智选车改变竞争格局，加速行业变革 |
| 混动→过渡路线仍占半壁 | 持续 | ★★☆ | 插混/增程在20万以下市场仍占主导，纯电占比50%+ |
| 车企垂直整合→降本深化 | 持续 | ★★☆ | 比亚迪CTB/一体化压铸/自研芯片持续降本 |

## 传导链

电动化渗透率>50% → 行业增速放缓 → 竞争焦点从"电动化"转向"智能化" → 智驾/座舱成核心差异化 → 车企研发投入从三电转向AI → 供应链格局重塑（Tier1向软件/芯片倾斜）→ 全球化出海打开第二空间

## 周期定位

**处于10→100的成熟期，但智能化开启了第二增长曲线。**

类比2011-2012年的智能手机市场——渗透率超过50%，增速放缓，但应用生态（App Store/Android Market）爆发，行业从"硬件红利"转向"生态红利"。

新能源车市场正处在类似的切换点——电动化的β减弱，智能化的β正在增强。

## 明星标的

- 整车龙头：比亚迪 / 理想汽车 / 小米集团 / 赛力斯
- 新势力：小鹏汽车 / 蔚来（淘汰赛中）
- 国企转型：长安汽车 / 吉利汽车 / 长城汽车
- 全球化标的：比亚迪（泰国/匈牙利/巴西）/ 奇瑞（未上市）
- 混动：比亚迪DM-i / 吉利雷神 / 长城Hi4
