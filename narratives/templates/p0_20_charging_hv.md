# 充换电/高压平台 — P0 #20

## 当前核心逻辑（主线）

**800V高压快充成为新能源车标配，"充电焦虑"取代"续航焦虑"成为核心矛盾。** 2025年20万以上新车型80%已搭载800V平台，快充桩功率从120kW向480kW演进。SiC器件在OBC/DC-DC中渗透率加速提升（见#6），液冷超充枪/高压连接器/薄膜电容等配套环节同步受益。

V2G（车网互动）是长期方向——电动车作为分布式储能参与电网调峰，2026年试点范围扩大，充电运营商的商业模式从"卖电"升级为"电力交易"。

## 旧逻辑（正在弱化）

充电桩是"配套"——跟着电动车销量走，充电桩运营是"重资产+低利润"，靠充电服务费根本赚不到钱。换电是蔚来的"独家"模式，其他车企不跟进。充电标准不统一，用户体验差。

现在800V平台已是新车型标配，快充从"加分项"变成"必选项"。充电运营商的盈利模型正在改善——峰谷价差扩大+光储充一体化+V2G电力交易，使充电站从"成本中心"变成"利润中心"。宁德时代/中石油/中石化等巨头入局换电/超充网络，基础设施投资加速。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 800V高压→SiC/OBC/DC-DC升级 | 当前主线 | ★★★ | 80%新车型搭载，功率器件量价齐升 |
| 液冷超充→充电功率480kW+ | 并行 | ★★★ | 华为600kW超充站布局，液冷枪线/模块受益 |
| 充电运营→V2G电力交易 | 早期 | ★★☆ | 峰谷价差扩大+光储充一体化，盈利模式改善 |
| 换电模式→宁德时代/中石油入局 | 并行 | ★★☆ | 蔚来+宁德时代+中石油换电联盟，标准化推进 |

| 薄膜电容/磁件→DC-DC/逆变器 | 持续 | ★★☆ | 800V平台对电容/磁件要求更高，价值量提升 |

## 传导链

新能源车渗透率提升 → 补能需求暴增 → 800V高压平台标配化 → 快充功率从120kW→480kW+ → 充电桩/车端功率器件同时升级 → SiC/OBC/连接器/电容全产业链受益 → 充电站从"电费"→"电力交易"模式升级

## 周期定位

**800V处于从1→10的爆发中段（已标配化），超充网络处于从0→1→1→10过渡。**

充电运营处于盈利拐点附近——2025-2026年是充电运营商从亏损→盈利的转折期。
