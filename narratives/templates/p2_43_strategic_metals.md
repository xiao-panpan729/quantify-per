# 小金属/战略金属（钨/钼/锑/镓/锗/稀土）— P2 #43

## 当前核心逻辑（主线）

**中国对战略金属的出口管制重塑全球供应链，战略价值重估。** 镓/锗（2023年管制）、锑（2024年管制）、钨/稀土（持续管控）——中国在这些战略金属的全球供应中占据主导地位（镓~94%、锗~80%、锑~48%、稀土~60%）。出口管制不是为了禁售，而是为了提升定价权和产业链控制力。

镓/锗是半导体和军工的关键原料（GaAs/GaN衬底、红外光学），锑用于阻燃剂/穿甲弹/光伏玻璃澄清剂，钨是硬质合金/军工穿甲弹的核心材料，稀土是永磁电机/风电/新能源汽车/HDD的核心材料。

## 旧逻辑（正在弱化）

小金属是"小众标的"——市场容量小（镓全球~300吨，锗~200吨），流动性差，价格波动大但缺乏持续的逻辑驱动。投资策略是"买涨价题材"。

现在：出口管制+AI/新能源/军工的新需求使战略金属从"小众周期品"升级为**"大国博弈的战略资源"**。中国在镓/锗/锑/稀土等品类中具有绝对供给优势，管制政策使这些品种的价格中枢有望系统性上移。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 出口管制→镓/锗/锑价格中枢上移 | 当前主线 | ★★★ | 中国管制+海外短期无法替代，战略金属价值重估 |
| 稀土→永磁电机+风电+人形机器人 | 持续 | ★★★ | 氧化镨钕是稀土永磁的核心，需求受益于新能源和人形机器人 |
| 钨→硬质合金+军工 | 持续 | ★★☆ | 中国钨储量/产量全球第一，硬质合金刀具需求稳定增长 |
| 钼→合金钢+油气管线 | 持续 | ★☆☆ | 跟随制造业/能源周期，弹性不大 |
| 锑→光伏玻璃+阻燃剂 | 并行 | ★★☆ | 光伏澄清剂是锑的新增量，需求增长20%+ |

## 传导链

中美博弈→出口管制升级 → 战略金属供给收紧 → 海外短缺→价格上行 → 中国矿企受益→战略金属价值重估 → 稀土永磁（新能源/机器人/风电）受益于下游需求增长

## 周期定位

**战略金属处于从1→10的价值重估期——出口管制是核心催化剂，需求（新能源/AI/军工）提供了底层的需求支撑。** 与周期的贵金属不同，战略金属的上涨更多是"中国供给管制驱动"。
