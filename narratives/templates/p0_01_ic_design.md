# IC设计（逻辑/SOC/ASIC/存储/CIS）— P0 #1

## 当前核心逻辑（主线）

AI端侧推理是最大增量。大模型从云端向终端迁移（手机/PC/汽车/IoT），端侧NPU和存算一体架构成为SoC升级的核心驱动力。RISC-V在IoT领域加速替代ARM，改变IP授权格局。

## 旧逻辑（正在弱化）

"摩尔定律放缓 + 国产替代"是前两年的主线叙事。国产替代已从情绪驱动进入业绩验证期；摩尔定律放缓的故事被Chiplet和先进封装部分"破解"，不再是核心矛盾。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| AI端侧推理→SoC升级周期 | 当前主线 | ★★★ | 全球性趋势，覆盖面广 |
| 存储芯片周期复苏 | 并行 | ★★☆ | HBM/NAND周期底部回升，AI拉动 |
| 国产替代→业绩验证 | 旧→过渡 | ★★☆ | 从主题转向业绩，个股分化开始 |
| RISC-V替代ARM | 早期 | ★☆☆ | 长期逻辑，短期渗透率提升慢 |
| CIS/模拟芯片复苏 | 边缘 | ★☆☆ | 跟随消费电子周期，弹性有限 |

## 传导链

AI应用爆发 → 端侧推理需求↑ → SoC算力要求↑ → 芯片设计复杂度↑ → 流片成本↑ → 头部集中（强者恒强）→ 先进制程产能吃紧 → 封测/EDA/IP受益

## 周期定位

当前处于**从1→10的爆发中段**。

从0→1已完成：AI大模型参数竞赛 → 云端训练芯片先受益。
第二阶段：推理从云端走向终端，驱动SoC设计范式转型。
类比2010年智能手机普及对IC设计的重塑——当时催生了ARM生态和移动SoC格局，这次是AI端侧重构。
