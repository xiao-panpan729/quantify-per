# 储能/氢能/新型电力 — P1 #34

## 当前核心逻辑（主线）

**储能从"配套"变为"刚需"——光伏配储+独立储能+AIDC备电三线驱动。** 中国新型储能装机2025年预计~100GWh，2026年有望~150GWh（同比增长50%）。储能商业模式在2025年逐步确立——独立储能可通过容量租赁+现货价差套利+辅助服务获得确定性收益。

氢能处于从0→1的政策驱动期——2025年氢能"十城千辆"示范推广，电解槽制氢成本在风光资源丰富地区已接近平价灰氢。但氢能产业链（储运/加氢站/燃料电池）仍不成熟。

## 旧逻辑（正在弱化）

储能="光伏配储的负担"——强制配储但没有收益模式，储能设备是"成本项"，投资逻辑是"谁有订单谁涨"。氢能是"远期故事"——技术不成熟，成本高，没有落地场景。

现在：独立储能的收益模型跑通了（容量租赁+现货价差）。光伏配储从"强制"变为"自驱"——光伏配储后可以提高上网电价。氢能的电解槽（PEM/碱性）技术成熟度提升，风光制氢项目的经济性在改善。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 大储/独立储能→装机50%+增长 | 当前主线 | ★★★ | 2026年储能装机有望~150GWh，商业模式逐步跑通 |
| AIDC备电→储能新增量 | 并行 | ★★★ | 见#31电池，AIDC备电是2025-2028年最确定的增量 |
|工商储→峰谷价差套利 | 并行 | ★★☆ | 峰谷价差拉大→工商储IRR改善→需求启动 |
| 氢能（电解槽）→风光制氢示范 | 早期 | ★★☆ | 风光制氢成本接近平价，但储运消纳仍是瓶颈 |
| 燃料电池→重卡/物流车示范 | 早期 | ★☆☆ | 氢能重卡在港口/矿区示范，基础设施不足限制放量 |
| 全钒液流→长时储能技术 | 萌芽 | ★☆☆ | 4-10小时储能时长，安全性好但成本高，2027年可能突破 |

## 传导链

风光装机↑ → 电网消纳压力↑ → 强制/自驱配储→储能装机持续增长 → 独立储能收益模式建立→储能从"成本"变"资产"→ 长时储能（液流/压缩空气/氢）需求在2027年后启动

## 周期定位

**电化学储能处于从1→10的加速爆发期（类似2021-2022年光伏的感觉）。** 氢能处于从0→1的示范导入期，商业化至少还需3-5年。

## 明星标的

- 储能电池：宁德时代 / 比亚迪 / 亿纬锂能 / 瑞浦兰钧（见#31）
- 储能逆变器（PCS）：阳光电源 / 锦浪科技 / 盛弘股份 / 上能电气
- 储能系统集成：海博思创 / 中车株洲所 / 比亚迪
- 温控/消防：英维克 / 高澜股份 / 国安达
- 电解槽：隆基氢能 / 阳光氢能 / 派瑞氢能
- 氢能储运：中材科技 / 京城股份 / 雪人股份
- 燃料电池：亿华通 / 潍柴动力 / 美锦能源
