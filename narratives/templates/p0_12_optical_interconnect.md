# 光互联/高速互联（CPO/光模块）— P0 #12

## 当前核心逻辑（主线）

**AI集群互联带宽需求每代翻倍，光互联是算力集群的"血管"。** 英伟达Rubin架构NVL576互联带宽需求从B200的1.8TB/s提升到3.6TB/s+，传统可插拔光模块的功耗/密度/成本难以支撑，CPO（共封装光学）从"实验室技术"进入"工程验证"阶段。

800G光模块2025年开始放量，1.6T是2026年的主战场。中国光模块企业在全球占据主导地位（中际旭创/新易盛/天孚通信合计份额>50%），是AI产业链中**少数中国已领先而非追赶的环节**。

## 旧逻辑（正在弱化）

光模块是"通信周期股"——跟随运营商/云厂商的资本开支周期波动，400G→800G→1.6T的升级是"渐进式"的。CPO被认为是5-10年后的远期技术。

现在AI集群互联需求的爆发速度远超传统数据中心升级周期，800G/1.6T的升级节奏从"3-4年一代"压缩到"1.5-2年一代"。CPO因为功耗和密度瓶颈提前到2026-2027年进入商用导入期。

## 传导链

AI模型参数↑ → 集群互联带宽↑ → 光模块速率从400G→800G→1.6T升级 → 功耗密度↑ → 可插拔模块受限 → CPO成为必然路径 → 光引擎/FAU/光纤阵列等新供应链形成 → 国内光模块保持全球份额

## 周期定位

**光模块处于从1→10的爆发中段。** 800G持续放量，1.6T周期启动在即。全球光模块市场2025年~200亿美元，AI驱动CAGR 25-30%。

CPO处于从0→1向1→10过渡的关键期——技术路线（标准CPO vs 近封装光学NPO）尚未统一，英伟达/台积电/Intel各自押注不同路线，供应链格局未定。

中国光模块企业的全球领先地位是AI产业链中最确定的结构性优势之一。

## 明星标的

- 光模块：中际旭创 / 新易盛 / 天孚通信 / 光迅科技 / 华工科技
- CPO：中际旭创（布局）/ Fabrinet / 台积电（3D封装光学）
- 硅光芯片：Silicon Photonics（Intel）/ Luxtera / 赛微电子
- 光引擎/FAU：天孚通信 / 仕佳光子
- 光芯片（EML/VCSEL）：源杰科技 / 长光华芯 / 武汉敏芯
