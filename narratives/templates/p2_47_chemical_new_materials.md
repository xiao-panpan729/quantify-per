# 化工新材料（氟/磷/有机硅/维生素）— P2 #47

## 当前核心逻辑（主线）

**AI算力需求外溢带动化工新材料——MLCC缺货带动PTFE需求、AIDC液冷带动氟化液需求是当前最明确的变量。** PTFE（聚四氟乙烯）是MLCC关键原材料（见#8），AI服务器MLCC用量是普通服务器的3倍+，PTFE在高速背板/高频CCL中的需求也在同步增长。

氟化工受制冷剂（HFCs）配额约束和AIDC液冷（氟化液冷却液）新场景双击——2025年HFCs配额制使供给收紧，价格上行；氟化液是液冷方案中的冷却介质（3M Novec的国产替代），在AIDC液冷中需求快速增长。

磷化工和有机硅跟随周期底部，需求端看不到类似AI催化的爆发性增量。维生素（VA/VE/VD3等）是典型的周期品——价格在底部，但供给出清后有望回升。

## 旧逻辑（正在弱化）

化工是"周期股"——看油价、看产能扩张、看下游需求（地产/汽车/纺织）。PTFE/氟化工是"空调制冷剂+不粘锅涂料"，与AI毫无关系。

现在：AI+新能源给化工新材料带来了成长性——PTFE从"不粘锅材料"升级为"AI服务器关键原料"，氟化液从"电子清洗剂"升级为"AIDC液冷核心介质"。化工不再是单纯的周期板块，细分龙头的成长属性在增强。

## 传导链

### 主线A：AI算力→MLCC→PTFE（既有）
AI数据中心扩张 → 服务器出货量↑ → MLCC需求3x+ → PTFE作为MLCC关键原料+高速CCL材料 → MLCC/PCB产能扩张→PTFE供应趋紧 → AIDC液冷→氟化液需求同步增长 → 氟化工全链条受益

### 主线B：霍尔木兹→油价→化工成本推升（2026年6月新增）
霍尔木兹海峡关闭 → 全球LNG贸易~20%受阻 → Brent油价中枢$95-100+ → 化工成本全面推升（打破"80美金以上无法传导"旧框架） → 海外供给缺口→中国化工出口需求旺盛 → 化工品价格重新定价 → 制冷剂/氟化工/煤化工等子行业EPS上修

### 支线：日本供应链风险→光刻胶/WF6等细分品种
霍尔木兹→油价上行→日本全面依赖能源进口→生产成本飙升→日本化工品供给收缩→光刻胶/特种气体(六氟化钨等)短缺风险 → 中国替代机遇（详见 [[transmission_chains.md]]）

## 周期定位

**PTFE处于从0→1→1→10过渡（AI服务器驱动）。氟化液处于从0→1的进口替代导入期。制冷剂处于从1→10的供给约束修复期。油价推升线处于"预期发酵→基本面验证"的过渡段——杨晖判断市场低估油价上行风险，新一轮配置行情正在靠近。**
