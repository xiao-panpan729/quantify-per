# 先进封装（T布/ABF载板/玻璃基板/TSV）— P0 #4

## 当前核心逻辑（主线）

**AI芯片算力需求突破物理极限，先进封装成为延续摩尔定律的核心路径。** 单芯片SoC的制程微缩已逼近物理极限（3nm以下成本暴增、回报递减），Chiplet架构将大芯片拆解为多颗小芯片通过先进封装互联，成为算力继续提升的必由之路。

T布（玻璃纤维布）是先进封装载板的**物理地基**——FCBGA ABF载板中T布成本占比约5-10%，但其均匀性/平坦度直接决定载板良率和信号完整性。AI芯片（GPU/ASIC）尺寸越来越大（2000mm²+），对T布的均一性要求从"工业级"跃升至"半导体级"，供需缺口开始暴露。

## 旧逻辑（正在弱化）

过去封装是半导体产业链的"后道"——技术含量低、利润率薄、跟随IC设计走。先进封装被看作是"可选升级"，大部分芯片用传统引线键合即可。现在Chiplet+先进封装从"可选"变成**AI芯片的必选项**，封装不再是下游配套，而是决定芯片能否量产的前置条件。

T布过去是成熟纺织工业品——电子布行业跟随PCB周期波动，产能过剩时价格战惨烈。现在AI服务器对高速材料的需求使T布从"大宗工业品"向**"半导体耗材"升级**。

## 传导链

AI算力竞赛 → 芯片集成度突破单芯片极限 → Chiplet架构普及 → 芯片尺寸增大/互联密度提升 → 先进封装从可选变必选 → ABF载板/T布/TSV等关键材料工艺吃紧 → 供应链价值从设计向封装载板转移 → 国产封装材料/载板迎来导入窗口

## 周期定位

**整体处于从1→10的爆发中段。**

从0→1已完成：台积电CoWoS已大规模量产，Chiplet在AI芯片全面采用。
当前催化剂：英伟达Rubin/B200出货 → 载板/T布需求跳升 → 供需缺口的确认。

T布子环节处于**从0→1向1→10过渡**的关键阶段：
- 过去T布是PCB周期品，中国企业做低端大宗
- AI服务器使高端T布（低开纤/高均一）需求从2024年开始井喷
- 2025年半导体级T布供不应求，进入"谁有产能谁受益"阶段
- 国内宏和科技等开始切入，但半导体级认证周期长（1-2年）

**类比**：2017年MLCC超级周期的前夜——需求结构突变，供给端反应滞后，价差利润弹性极大。

## 关键数据锚点

- AI服务器T布用量：2025年963万米 → 2028年6959万米（97%+ CAGR 2026-2027）
- 全球T布供给缺口：2025年14% → 2026年18% → 2027年8%（因扩产释放逐步收窄）
- T布在FCBGA ABF载板成本占比5-10%，在FCCSP BT载板占比3-5%
- 国内T布产能占全球约30%，但半导体级产品占比极低
