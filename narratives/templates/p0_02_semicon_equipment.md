# 半导体设备（光刻/刻蚀/薄膜/检测）— P0 #2

## 当前核心逻辑（主线）

**双重国产替代加速**：美国MATCH法案要求日荷对齐出口管制，过去"去美化"现在叠加"去日化"。国内存储厂商（长鑫/长存）批量扩产已启动，先进逻辑扩产存在超预期可能，先进封装即将放量。国产化率从2018年4.91%→2024年18.02%，进入快速爬升阶段。

## 旧逻辑（正在弱化）

以前半导体设备是"跟着晶圆厂产能周期走"——全球半导体资本开支上行→设备受益；下行→设备承压。现在国产替代从"可选"变成"必选"，MATCH法案使外购路径进一步收窄，设备板块从周期品变成**自主可控的刚需品**。估值锚也从PE转向国产替代空间。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 存储扩产→国产设备批量导入 | 当前主线 | ★★★ | 长鑫/长存扩产确定性最强 |
| MATCH法案→去日化加速 | 催化剂 | ★★★ | 光刻机/刻蚀/薄膜国产化紧迫性提升 |
| 先进逻辑扩产→超预期可能 | 并行 | ★★☆ | 华虹等扩产节奏可能加快 |
| 零部件国产替代→弹性大于整机 | 并行 | ★★☆ | 国产化率低，放量弹性大 |
| 后道设备（测试）→AI驱动 | 持续 | ★★☆ | AI芯片测试需求爆发，海外龙头交期长 |
| 先进封装设备→新增量 | 早期 | ★★☆ | 盛合晶微已上市，扩产即将放量 |

## 传导链

美国出口管制加码（MATCH/华虹断供）→ 国内晶圆厂加速国产设备验证导入 → 存储/逻辑/先进封装三大扩产同步启动 → 设备国产化率快速提升 → 零部件"卡脖子"环节成为瓶颈 → 零部件国产替代弹性大于整机

## 周期定位

处于**从1→10的加速期**。

从0→1已完成：国产设备在成熟制程已批量量产，部分进入先进制程验证。
当前催化剂：MATCH法案+长鑫IPO+存储扩产三重共振，2026年下半年扩产存在超预期可能。
国产化率从18%向30%+的爬升过程中，设备板块的投资逻辑从"主题炒作"转向"业绩验证"。

**关键时间节点**：2026H2扩产预期 + 150天MATCH对齐窗口。
