# AI大模型/多模态/Agent — P0 #9

## 当前核心逻辑（主线）

**从"模型竞赛"到"应用落地"，Agent是2026年的最大变量。** 2023-2025年是基座大模型的参数竞赛期（GPT-4→Claude 3→Gemini→DeepSeek-R1），2026年行业主线已切换为：推理成本快速下降（DeepSeek-R1训练成本降至GPT-4的1/10）→应用场景爆发→Agent自主决策成为落地关键形态。

多模态（图像/视频/音频/代码）是模型能力的第二曲线。文本模型的API价格已接近归零（降幅99%+），多模态应用是差异化竞争的核心方向。

## 旧逻辑（正在弱化）

"训练算力需求无限，买GPU就赢"是过去两年的主线。市场关注的是：谁的GPU多、谁的参数大、谁的训练集群强。Scaling Law被认为是铁律，模型能力随参数/数据/算力线性增长。

现在Scaling Law已出现边际递减（特别是文本模型），推理算力占比从20%提升到60%+，算力需求结构从"训练主导"转向"推理主导"。市场关注点从"谁的模型强"转向"谁的应用能赚钱"。DeepSeek证明了小团队也能训出顶级模型，参数竞赛不再是唯一路径。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 推理成本下降→应用爆发 | 当前主线 | ★★★ | 推理成本降到1/100，激活长尾场景，Agent/工作流/AI员工 |
| Agent→自主决策+工具调用 | 并行 | ★★★ | 2026年的核心产品形态，Manus/Cognition/各种Agent框架 |
| 多模态→图像/视频/音频 | 并行 | ★★★ | 文本模型同质化，多模态是差异化方向，Sora/GPT-4o |
| 模型开源→生态分裂 | 持续 | ★★☆ | DeepSeek/Meta LLaMA开源模型缩小与闭源差距 |
| AI编程→开发者效率革命 | 持续 | ★★☆ | Claude Code/GitHub Copilot/Cursor改变软件开发方式 |
| AI基础设施建设→算力需求 | 旧→过渡 | ★★☆ | 仍在增长但增速放缓，从训练→推理结构切换 |

## 传导链

模型能力接近实用阈值 → 推理成本断崖下降 → 开发者涌入构建应用 → Agent框架成熟（规划+记忆+工具调用）→ 企业级AI部署加速 → 推理算力需求爆发 → 芯片/服务器/数据中心需求结构变化（推理占比提升）

## 周期定位

**大模型处于从1→10的应用爆发前夜。** 类比2010年智能手机爆发前的移动互联网——基础设施已完成（模型能力到位），成本已降到可商用（推理价格），但杀手级应用还在早期。

Agent形态类比2007年iPhone的触摸屏交互——新的交互范式（从对话→自主决策）将是下一波应用爆发的核心驱动。
