# 大宗有色金属（铜/铝/锌/铅/锡）— P2 #44

## 当前核心逻辑（主线）

**铜是"AI+能源转型"双驱动下的最受益基础金属。** 铜的逻辑来自：AI数据中心（单AIDC用铜~5000吨，年需求增量~50万吨+）、新能源车（单车用铜~80kg，燃油车~20kg）、光伏风电（每GW用铜~5000吨）。供给端约束明确——全球铜矿品位下降+ESG审查+新矿投产周期8-10年，在产矿山产量不增反降。铜是供给约束最强的基本金属。

铝的逻辑类似但弱于铜——供给受限（国内电解铝产能天花板~4500万吨）、需求受益于光伏边框+新能源车轻量化。铅/锌跟随全球工业周期，无独立增量逻辑。锡受益于AI芯片封装（焊料）+光伏焊带的需求增长。

## 旧逻辑（正在弱化）

有色金属是"周期股"——价格波动大，跟随全球PMI/中国房地产/基建投资周期。铜是"铜博士"（经济晴雨表），铝跟着房地产走。

现在：铜的需求结构从"房地产+基建"切换为"AI+新能源+电动车"——铜已经从"旧经济"的铜变为"新经济"的铜。供给约束的刚性（新矿不出+老矿减产）使铜价中枢系统性上移。铜企不再被给周期股估值（8-10x PE），而应向成长股估值切换（15-20x PE）。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 铜→AI+新能源+电动车三驱动 | 当前主线 | ★★★ | 精炼铜消费增速从~2%提升到~4%，结构性缺口扩大 |
| 铜供给约束→新矿不出+老矿减产 | 并行 | ★★★ | 全球铜矿产能利用率下降，2025-2028年缺口持续扩大 |
| 铝→供给天花板+新能源需求 | 持续 | ★★☆ | 国内电解铝产能已接近天花板，光伏+汽车需求支撑价格 |
| 锡→AI芯片封装增量 | 并行 | ★★☆ | AI芯片封装（焊料）+光伏焊带，锡焊料是半导体互联的核心 |
| 锌/铅→跟随工业周期 | 弱势 | ★☆☆ | 无独立增量逻辑，但供给端也在收紧 |
| 铜矿并购→行业集中度提升 | 持续 | ★★☆ | 紫金/洛钼海外并购扩张，中国铜企全球份额提升 |

## 传导链

AI基础设施+新能源+电动车 → 铜/铝/锡需求结构性增长 → 铜矿供给刚性约束（无新矿）→ 供需缺口持续扩大 → 铜价中枢上移 → 铜矿企业利润弹性（铜价每涨10%，利润涨30-50%）→ 估值从周期PE→成长PE切换

## 周期定位

**铜处于从1→10的结构性缺口周期（2025-2028年），铝处于从1→10的供给约束周期。** 铜是基本金属中最具"成长性"的品种——不是因为需求爆发，而是因为供给受限+需求结构升级。

## 明星标的

- 铜：紫金矿业 / 洛阳钼业 / 铜陵有色 / 江西铜业 / 西部矿业
- 铝：中国铝业 / 云铝股份 / 神火股份 / 南山铝业
- 锡：锡业股份 / 兴业银锡
- 锌/铅：中金岭南 / 驰宏锌锗 / 株冶集团
