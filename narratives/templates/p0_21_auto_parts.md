# 汽车零部件 — P0 #21

## 当前核心逻辑（主线）

**中国汽车零部件企业的全球化进程加速，从"中国制造"→"中国品牌"→"全球Tier1"。** 中国新能源车产业链的规模优势（全球60%+新能源车产自中国）催生了一批具有全球竞争力的零部件企业——三电系统（宁德时代/汇川技术）、热管理（三花智控/拓普集团）、轻量化（旭升集团/爱柯迪）、底盘（伯特利/中鼎股份）。

全球化是核心增量——中国零部件企业凭借"性价比+交付速度"进入海外车企供应链（特斯拉/大众/宝马/奔驰），海外业务毛利率通常比国内高5-10个百分点。

## 旧逻辑（正在弱化）

汽车零部件是"代工"——低利润率、依赖大客户、周期波动大。中国零部件企业做低端（轮毂/玻璃/内饰），高端零部件（电控/底盘/热管理）被博世/大陆/采埃孚/电装等外资Tier1垄断。

现在中国零部件企业在三电/热管理/轻量化/智驾等核心赛道已具备全球竞争力。全球汽车供应链正在重塑——外资Tier1在电动化转型中被"降维打击"，中国零部件企业以"响应速度+成本优势+技术迭代"抢占份额。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 全球化→中国零部件出海 | 当前主线 | ★★★ | 海外营收占比持续提升，特斯拉/大众供应链是中国企业的桥头堡 |
| 三电系统→电池/电驱/电控 | 持续 | ★★★ | 宁德时代全球份额第一，汇川电驱持续渗透 |
| 热管理→集成化升级 | 并行 | ★★☆ | 从散热器→热泵系统，单车价值量从500→3000元 |
| 轻量化→一体化压铸 | 并行 | ★★☆ | 一体化压铸降本30%，渗透率快速提升 |
| 底盘/制动→线控化 | 早期 | ★★☆ | 线控底盘是智驾执行端核心，伯特利等突破 |
| 内外饰→低估值稳定基本盘 | 边缘 | ★☆☆ | 成熟行业，随车市波动，非弹性来源 |

## 传导链

中国新能源车全球份额↑ → 国产零部件同步成长 → 规模效应+技术迭代 → 进入全球车企供应链 → 海外营收占比↑ → 海外毛利率高于国内 → 利润弹性大于收入弹性 → 估值从"国内零部件"→"全球Tier1"切换

## 周期定位

**三电/热管理处于从1→10的全球化加速期，轻量化/线控底盘处于从0→1→1→10过渡。**

中国汽车零部件企业的全球化空间相当于2015年家电龙头（美的/海尔）出海的早期阶段——海外份额10-20%，天花板在50%+。

## 明星标的

- 三电：宁德时代 / 汇川技术 / 方正电机 / 大洋电机
- 热管理：三花智控 / 拓普集团 / 银轮股份 / 中鼎股份
- 轻量化/一体化压铸：旭升集团 / 爱柯迪 / 文灿股份 / 广东鸿图
