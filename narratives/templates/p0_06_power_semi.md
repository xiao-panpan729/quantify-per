# 功率半导体（IGBT/SiC/GaN/第三代）— P0 #6

## 当前核心逻辑（主线）

AI数据中心电源革命驱动功率半导体需求结构性爆发。单颗AI芯片功率从250W飙至1000W+，单机柜向MW级迈进，推动供电架构从传统工频变压器向SST（固态变压器）/HVDC迭代。功率半导体在SST中成本占比30-40%，SiC MOSFET（耐压10-15kV）和GaN HEMT（高频DC/DC）是核心增量器件。

## 旧逻辑（正在弱化）

功率半导体过去的故事是**新能源车+光伏逆变器**双轮驱动。IGBT是主角，SiC是"未来"。现在AI数据中心成了第三条腿且增速最快——SiC从"未来"变成了"现在必须"，而传统工频变压器被SST取代的趋势一旦铺开，功率半导体的需求结构将发生根本性变化。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| AIDC电源→SiC/GaN爆发 | 当前主线 | ★★★ | 800V HVDC+SST架构驱动，功率半导体增量最大 |
| 新能源车→SiC替代IGBT | 持续 | ★★☆ | 仍是重要基本盘，但增速预期已被消化 |
| 光伏逆变器→IGBT需求 | 稳态 | ★★☆ | 跟随装机量，非弹性来源 |
| 国产替代→功率器件 | 并行 | ★★☆ | 国产SiC/IGBT进入验证期，但与国际差距仍在 |
| 消费电子快充→GaN | 边缘 | ★☆☆ | 量大有周期性，但单机价值量低 |

## 传导链

AI算力扩张 → 芯片功率提升（1000W+/chip）→ 机柜功率MW级 → 传统配电架构不满足 → SST/HVDC替代 → 功率半导体是核心瓶颈（成本30-40%）→ SiC/GaN需求爆发 → 产能紧平衡 → 设备/衬底受益

## 周期定位

处于**从1→10的爆发早期**。

从0→1已完成（SiC在新能源车已批量上车，AIDC方案已有样机）。
当前催化：英伟达800V供电白皮书+Rubin ultra量产节点→SST从试点走向小批量。
类比2019年光伏PERC电池从实验室走向量产——技术路线已通，规模效应未到，先行者受益。

**三个电压等级对应的功率半导体格局：**
- 13.8kV AC→800V DC（一次侧）：SiC MOSFET为主，耐压10-15kV
- 800V→50V DC/DC（二次侧）：GaN HEMT最优，开关频率1MHz+
- 50V→1V VRM（三次侧）：硅基MOSFET为主，Vicor/MPS方案
