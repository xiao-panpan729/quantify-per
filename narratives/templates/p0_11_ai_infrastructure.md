# 算力基础设施（服务器/液冷/数据中心）— P0 #11

## 当前核心逻辑（主线）

**AI算力需求从"训练驱动"切换为"推理驱动"，算力基础设施产业链结构正在重塑。** 推理算力占比从2024年的~20%预计提升到2027年的60%+。推理侧对功耗/延迟/成本更加敏感，推动AI服务器架构从"集中式训练集群"向"分布式推理节点"扩散，液冷从"可选"变成"刚需"——单机柜功率密度突破100kW，风冷无法满足。

中国AI服务器市场额外受益于国产化替代：英伟达H20被限制后，华为昇腾/海光DCU/寒武纪思元等国产算力芯片加速导入，服务器OEM格局随之洗牌。

## 旧逻辑（正在弱化）

算力基础设施="英伟达GPU+IDC机柜"。上一轮算力牛市的叙事是：谁有H100谁就赢，服务器厂商是"搬砖"的（低利润率组装），液冷是"概念"（渗透率<5%），数据中心是"REITs收租"。

现在：英伟达GPU供给不再稀缺（CoWoS产能释放+竞品出现），服务器厂商通过液冷/系统设计构建差异化，液冷渗透率2025年突破20%拐点，数据中心从"资源型"升级为"算力运营型"（AI工厂概念）。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 推理算力爆发→服务器需求结构变 | 当前主线 | ★★★ | 推理节点数量级高于训练集群，边缘推理是增量 |
| 液冷→高密度机柜刚需 | 并行 | ★★★ | 100kW+风冷不够用，液冷2025年20%→2027年50%+ |
| 国产算力→华为昇腾国产替代 | 并行 | ★★★ | H20被限后的唯一选择，生态（CANN）仍在完善 |
| AIDC（AI数据中心）→新型电力负荷 | 早期 | ★★★ | 单数据中心300MW+，电网接入成为瓶颈 |
| 服务器OEM→液冷差异化 | 持续 | ★★☆ | 普通组装利润<5%，液冷方案利润15%+ |
| 数据中心从"地产"→"算力运营" | 早期 | ★★☆ | 类似从商业地产→云计算的转型 |

## 传导链

AI模型能力↑ → 推理成本↓ → 应用爆发 → 推理算力需求↑ → 机柜功率密度↑ → 液冷渗透率↑ → AIDC单位投资额翻倍 → 服务器/液冷/配电/UPS全产业链升级

## 周期定位

**AI服务器处于从1→10的爆发中段，液冷处于从0→1向1→10过渡。**

液冷拐点已过（渗透率>20%），2026-2027是液冷放量的关键窗口，类比2018年光伏单晶替代多晶的转折点。

国产算力处于从0→1的验证期——华为昇腾910在部分场景已可替代A100，但CANN生态成熟度较CUDA仍有差距。

## 明星标的

- AI服务器：工业富联 / 浪潮信息 / 中科曙光 / 紫光股份
- 液冷：英维克 / 高澜股份 / 申菱环境 / 曙光数创
- AIDC：万国数据 / 世纪互联 / 秦淮数据
- 国产算力芯片：华为昇腾 / 海光信息 / 寒武纪
- 算力调度：中科曙光 / 首都在线
