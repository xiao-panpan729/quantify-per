# 信创/国产替代 — P1 #35

## 当前核心逻辑（主线）

**信创从"政策引导"进入"市场化+常态化"阶段。** 2025-2027年是信创替换的关键窗口——党政信创（县乡级替换）接近尾声，行业信创（金融/能源/运营商/教育/医疗）进入大规模招标期。行业信创的市场空间是党政信创的3-5倍，且采购方支付能力更强。

信创的核心驱动已从"国产化率达标"变为"产品力提升+国家信息安全"双轮驱动。华为鲲鹏/昇腾生态的成熟是最大变量——华为在硬件（CPU/GPU/服务器）+操作系统（欧拉）+数据库（高斯）+云的全栈能力使信创从"能用"变为"好用"。

## 旧逻辑（正在弱化）

信创是"炒政策"——党政PC/服务器的替代是一波流（~2000万台PC替换），替换完了就没有增量。国产CPU/OS/数据库"有功能无体验"，x86/Windows/Wintel生态的粘性太大，行业用户不愿意换。

现在：华为生态从"替代方案"变成了"主流选择"——鲲鹏在运营商/金融的出货量已相当可观，信创不再是"替不替"的问题而是"什么时候替"的问题。行业信创的采购体量和持续性远超党政市场。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 行业信创→金融/运营商/能源大规模招标 | 当前主线 | ★★★ | 三大运营商+六大行+两网的信创采购持续放量 |
| 华为生态→鲲鹏/昇腾/欧拉/高斯全栈 | 并行 | ★★★ | 华为从"备选"变"首选"，生态成熟度大幅提升 |
| 国产CPU→x86/ARM/龙架构 | 持续 | ★★☆ | 海光x86(性能最佳)/鲲鹏ARM(生态最好)/龙芯自主架构(政策) |
| 国产OS→统信/麒麟 | 持续 | ★★☆ | 桌面端Linux市占率已超10%，服务器端更高 |
| 数据库→分布式替代集中式 | 持续 | ★★★ | 达梦/人大金仓/华为高斯/ OceanBase在金融/运营商批量替代Oracle/DB2 |
| 信创云→多云/私有云 | 持续 | ★★☆ | 行业客户从VMware→华为云/浪潮云/移动云迁移 |

## 传导链

中美科技脱钩持续 → 信息安全要求提升 → 行业信创招标常态化 → 国产CPU/OS/DB/云产品力提升→华为生态全面成熟 → 信创从"补课"到"主流" → 上游(CPU/GPU/存储芯片)+下游(集成/运维/安全)全面受益

## 周期定位

**行业信创处于从1→10的加速爆发期（2025-2027年），党政信创处于10→100的收尾和运维阶段。**

核心驱动：华为鲲鹏/昇腾芯片在运营商/金融的出货量预计2026年翻倍。
