# 锂/钴/镍（新能源金属）— P2 #45

> 与 #32 锂资源 有重叠，此处聚焦三者的供需格局对比和产业链位置

## 当前核心逻辑（主线）

**新能源金属经历价格暴跌后，供需再平衡节奏各不相同——镍率先触底，锂接近底部，钴仍在寻底。** 三元电池（NCM/NCA）和磷酸铁锂（LFP）的路线之争决定了三种金属的长期需求格局——LFP市占率提升利好锂（磷酸铁锂需要碳酸锂）、利空钴/镍（三元正极不需要钴、镍用量也减少）。LFP全球市占率已从~30%提升到~70%，对钴/镍需求的结构性压制是长期的。

三种金属的最大区别在于供给约束的强度：锂供给充足（盐湖+锂矿）+钴供给高度集中（刚果金~70%）但回收率高+镍供给过剩（印尼镍铁产能大爆发）。

## 旧逻辑（正在弱化）

锂钴镍是"电池三兄弟"——电动车爆发带动三者同涨同跌，"家里有矿"就涨。市场关注的是"哪个品种供给最紧张/库存最低"。

现在三种金属的"角色分化"——锂的供需正在出清（见#32）、镍的过剩主要来自印尼（政策风险大）、钴面临LFP替代的长期结构性利空。投资逻辑从"买矿"变为**"买成本曲线最左侧+需求韧性最强的品种"**。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 锂→供需出清进行中，接近底部 | 当前主线 | ★★★ | 见#32锂资源，碳酸锂7万/吨接近成本支撑 |
| 镍→印尼供给过剩，不锈钢维持基本盘 | 并行 | ★★☆ | 印尼镍铁/HNPI产能释放，价格低位徘徊，三元/前驱体企业受益于低成本 |
| 钴→LFP替代压制+刚果金地缘风险 | 弱势 | ★★☆ | 钴在三元正极中的用量持续下降，但刚果金供给风险提供价格支撑 |
| 镍锰/无钴电池→远期技术路线 | 早期 | ★☆☆ | 宁德时代/松下/LG开发无钴电池，远期利空钴需求 |

## 传导链

LFP渗透率↑ → 钴需求↓ → 镍三元需求份额↓ → 锂需求相对受益 → 锂/钴/镍各自的供需再平衡 → 价格分化（锂修复/钴承压/镍过剩）→ 电池成本下降→电动车渗透率继续提升

## 周期定位

三种金属处于不同的周期位置：锂在出清底部（即将修复）、镍在产能过剩期（等待出清）、钴在需求萎缩+地缘风险并存的结构性矛盾期。

## 重点标的

- 锂（见#32）：赣锋锂业 / 天齐锂业 / 盐湖股份 / 永兴材料
- 钴：华友钴业 / 寒锐钴业 / 洛阳钼业（TFM铜钴矿）
- 镍：华友钴业（印尼镍项目）/ 格林美（镍回收+前驱体）/ 中伟股份（前驱体）
