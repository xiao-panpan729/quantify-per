# EDA/IP/设计服务 — P0 #5

## 当前核心逻辑（主线）

**Chiplet时代使EDA从"画图工具"升级为"系统设计平台"。** 传统EDA解决的是单芯片设计的问题，Chiplet+异构集成使设计空间从SoC内部的数百万门扩展到多Chiplet互联的数十亿门系统——EDA需要同时处理热/功耗/信号完整性/多物理场仿真，工具复杂度倍增，单点工具向平台级整合加速。

RISC-V在IoT和边缘计算的渗透改变IP授权模式——从ARM的高授权费模式转向开源+商业服务模式，IP行业的收入结构正在被重塑。芯原股份等设计服务商从"卖人力"向"卖IP+设计平台"转型，估值模型从PE向PS切换。

## 旧逻辑（正在弱化）

EDA是"小众软件"——全球市场~150亿美元，和新思科技/Cadence两巨头垄断，国产EDA在夹缝中生存。IP授权是"卖图纸"的稳定生意，ARM统治移动端，x86统治桌面/服务器。设计服务是"劳动密集型外包"，毛利率30-40%没有护城河。

现在这套三分类框架正在模糊：EDA公司开始提供IP和设计服务（新思收购Ansys做多物理场），IP公司向上游设计服务延伸，设计服务商自研IP建立壁垒。边界消失意味着**每个环节都有向全栈扩张的机会**。

## 传导链

AI芯片复杂度↑ → 单芯片SoC→多Chiplet系统 → EDA工具从"单点"向"多物理场平台"升级 → 单设计成本↑（先进制程流片成本$5000万+）→ 设计服务和IP复用成为降本关键 → RISC-V降低IP门槛 → 国产EDA/设计服务迎来替代窗口

## 周期定位

**EDA处于从1→10的稳定增长期，国产EDA处于从0→1向1→10过渡。**

全球EDA市场跟随半导体设计复杂度稳步增长（~8% CAGR），AI+chiplet是增量催化剂。
国产EDA处在特殊窗口：出口管制使国内晶圆厂/设计公司转向国产工具，但数字芯片全流程仍有差距。

IP行业：ARM生态仍坚固，但RISC-V在IoT/边缘已形成真实替代。类比2005年ARM刚进入手机市场时的早期渗透。

设计服务：国内厂商从"低端外包"向"高端定制"转型中，芯原股份的芯片设计平台（SiPaaS）是转型标杆。
