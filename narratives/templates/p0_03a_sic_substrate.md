# 半导体衬底材料（SiC/金刚石/硅/蓝宝石）— P0 #3a

> 附属于 #3 半导体材料，但衬底是独立供应链环节，跨多条产业链

## 当前核心逻辑（主线）

SiC衬底是功率半导体爆发的**限速环节**。SiC器件在AIDC电源、新能源车主驱中的渗透率快速提升，而衬底成本占SiC器件总成本40-50%，是降本和产能扩张的核心瓶颈。衬底从6英寸向8英寸转型是当前最大产业趋势——8英寸可切割芯片数提升80-90%，是SiC器件价格降至硅基IGBT可替代区间的关键路径。

## 旧逻辑（正在弱化）

以前衬底是"配角"——硅衬底是成熟大宗品，随半导体周期波动，没有独立叙事。SiC衬底也被当作"远期故事"，等SiC器件先放量再说。现在SiC器件已在AIDC电源和主驱中批量上车，衬底变成了**卡脖子环节**，从"跟随者"变成了"决定者"。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| SiC衬底→8英寸转型→降本曲线 | 当前主线 | ★★★ | 决定SiC渗透率能否加速的关键 |
| SiC衬底→AIDC电源需求爆发 | 并行 | ★★★ | 功率半导体的上游瓶颈，p0_06的延伸 |
| 金刚石衬底→AI芯片散热 | 早期 | ★★☆ | 2000W/m·K热导率，技术路线未定型 |
| 硅衬底→先进制程 | 稳态 | ★★☆ | 跟随逻辑/存储周期，无独立弹性 |
| 蓝宝石衬底→LED/MicroLED | 边缘 | ★☆☆ | 成熟市场，增长有限 |

## 传导链

SiC器件渗透率↑ → 衬底需求↑ → 6英寸产能吃紧 → 8英寸转型加速 → 衬底成本占比40-50%决定器件降价空间 → 衬底良率+缺陷密度是核心壁垒 → 衬底厂扩产节奏决定整个SiC产业链增速

## 周期定位

SiC衬底处于**从1→10的爆发早期**。

从0→1已完成：SiC衬底在新能源车主驱已批量供货，6英寸为主力。
当前催化剂：AIDC电源（800V HVDC）+ 新能源车渗透率继续提升 → 8英寸衬底需求启动。
8英寸衬底的产业节奏：2025年样品验证 → 2026年小批量 → 2027年规模量产。

类比2010年光伏硅片从6英寸→8英寸的转型——当时催生了中国的硅片霸主地位。

金刚石衬底处于**从0→1的萌芽期**。技术路线（CVD单晶/多晶/复合材料）未定，没有规模量产，但AI芯片散热瓶颈是刚性需求。
