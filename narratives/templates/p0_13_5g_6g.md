# 5G/6G通信 — P0 #13

## 当前核心逻辑（主线）

**AI+通信双向赋能。** 一方面，AI数据中心爆发拉动光互联/高速PCB/高频材料需求（与#11/#12/#8交叉）；另一方面，5G-A（5.5G）和6G的研发投入使通信设备产业链维持景气。5G-A在低空经济（通感一体化）和工业互联网（uRLLC增强）中是关键基础设施，6G的THz频段和智能超表面（RIS）技术预研已启动。

但在整体P0优先级中，通信板块的独立弹性低于AI/半导体/新能源——更多是"跟随受益"而非"核心驱动"。

## 旧逻辑（正在弱化）

"5G换机潮+基站建设"是2019-2021年的大主线。运营商大规模建站→设备商受益→手机换机→消费电子受益。5G是"万物互联的基础设施"。

现在5G基站建设高峰已过，运营商资本开支总量平稳（从5G峰值下行，向算力网络转移）。万物互联的故事被AI替代——真正的增量不在5G连接数，而在AI驱动的算力网络。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 5G-A通感一体化→低空经济网络 | 并行 | ★★☆ | 低空经济的前提条件，需运营商升级网络 |
| 6G预研→THz/智能超表面/星地融合 | 早期 | ★★☆ | 2030年商用，当前是标准制定+技术储备期 |
| AI数据中心→高速互联需求 | 交叉 | ★★★ | 见#12光互联，#11算力基础设施 |
| 通信设备→国产替代 | 持续 | ★★☆ | 华为/中兴全球份额稳定，高端芯片仍受限 |
| 运营商→算力网络转型 | 持续 | ★★☆ | 移动/电信/联通从管道商向算力服务商转型 |

## 传导链

AI应用爆发 → 算力需求 → 数据中心互联带宽升级 → 通信设备速率提升 → 产业链上游（PCB/CCL/光模块/连接器）受益；同时低空经济→5G-A通感一体化→基站设备/天线升级

## 周期定位

**5G处于成熟期（10→100），5G-A处于从0→1导入期，6G处于从0→1的萌芽期。**

5G建站高峰已过，塔顶设备进入存量替换/升级周期。
5G-A是增量（通感一体化是低空经济刚需），但运营商投资节奏不确定。
6G商业化至少到2030年，当前主要是设备商的研发布局（华为6G研发投入>5G同期）。

## 明星标的

- 通信设备：华为（未上市）/ 中兴通讯 / 烽火通信
- 天线/射频：信维通信 / 硕贝德 / 通宇通讯
- 基站PCB/CCL：沪电股份 / 深南电路 / 生益科技
- 光模块/光传输：中际旭创 / 新易盛 / 光迅科技
- 运营商：中国移动 / 中国电信 / 中国联通
