# 低空经济（eVTOL/无人机/飞行汽车）— P0 #26

## 当前核心逻辑（主线）

**低空经济从"政策驱动"进入"产业落地"阶段。** 2024年是低空经济"元年"（中央经济工作会议定调，20+省市出台政策），2025-2026年进入适航认证和试点运营期。亿航EH216-S已获三证（TC/PC/AC），峰飞/时的/沃兰特等eVTOL企业进入TC适航认证阶段。

无人机物流/配送是当前落地最快的场景——顺丰丰翼/美团无人机已开通100+条常态化航线，2025年日均配送量突破万单。农业植保无人机已是成熟市场（年作业面积20亿亩次）。

低空基础设施（通感一体化基站/低空智联网/起降场/空管系统）是前提条件，2025-2027年是基础设施集中建设期，投资确定性高于飞行器本身。

## 旧逻辑（正在弱化）

低空经济是"炒政策概念"——没有商业模式，没有适航标准，没有基础设施。eVTOL是"富人的玩具"，距离大众市场遥遥无期。空域管制（中国空域90%管制）是根本性制约。

现在：适航标准体系逐步建立（中国已走在全球前列，eVTOL适航审定进度领先欧美），深圳/合肥/成都等试点城市低空空域逐步开放。eVTOL的商用路径清晰——"货运→载人→大众出行"三步走。TCO测算显示eVTOL的单位成本已接近地面专车。

## 传导链

政策框架确立（中央+地方）→ 适航标准体系建立 → 空域逐步开放 → 基础设施（5G-A/起降场/空管）开建 → eVTOL/无人机企业获取运营资质 → 试点商业运营（旅游/物流）→ 运营数据积累→成本曲线下降 → 大众出行场景开放 → 产业链全面爆发

## 周期定位

**eVTOL处于从0→1向1→10过渡的适航认证期。** 类比2015年电动车——车型已出、政策支持、商业化未到，但产业链布局的最佳窗口。

无人机物流处于从1→10的放量期——商业模式已跑通，2025-2027年从区域试点→城市级覆盖。

低空基建处于从0→1→1→10过渡——订单已出现但尚未大规模招标。

## 明星标的

- eVTOL整机：亿航智能 / 峰飞航空（未上市）/ 时的科技 / 沃兰特 / 小鹏汇天
- 无人机物流：美团无人机 / 顺丰丰翼 / 京东物流
- 低空基建/空管：莱斯信息 / 川大智胜 / 四创电子 / 国睿科技
- 通感一体化基站：中兴通讯 / 华为 / 信科移动
- 复材/结构件：光威复材 / 中复神鹰 / 楚江新材
- 动力/三电：宁德时代（凝聚态电池）/ 汇川技术（电驱）
