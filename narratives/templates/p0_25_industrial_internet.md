# 工业互联网/工业软件 — P0 #25

## 当前核心逻辑（主线）

**AI+工业软件正在重塑制造业的研发/生产/运维全流程。** CAD/CAE/EDA/PLM等工业软件在设计端受益于AI增强（AI辅助设计→生成式CAD/自动仿真）；MES/SCADA/WMS在制造端受益于工业互联网平台整合；预测性维护/数字孪生在运维端降本增效。

工业软件的核心驱动是：**中国工业软件国产替代（从~10%→30%+）的确定性进程**。研发设计类（CAD/CAE/EDA）的国产化率最低（~5-10%），但也是政策扶持力度最大的环节。生产管理类（MES/SCADA）的国产化率相对较高（~30%），竞争激烈但市场空间大。

## 旧逻辑（正在弱化）

工业软件是"小众板块"——全球市场被达索/Siemens/Ansys/新思科技等垄断，中国工业软件公司在夹缝中生存，产品力差距大、"有功能无体验"。投资热度低，流动性差。

数字孪生/工业互联网是"概念"——企业上ERP/MES是"面子工程"，真实价值有限。制造业数字化投入意愿低，ROI不清晰。

现在：AI大模型使工业软件的交互方式从"专业操作"→"自然语言"——工程师可以用自然语言描述需求，AI自动生成CAD模型/仿真设置/代码。出口管制使国产替代从"政策引导"变为"生存必需"。制造业劳动力成本上升加速数字化投入。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| 国产替代→研发类工业软件 | 当前主线 | ★★★ | 中望CAD/华大九天/概伦电子在各自领域突破 |
| AI+工业软件→设计范式变革 | 并行 | ★★☆ | 生成式AI辅助设计，降低使用门槛，扩大市场 |
| MES/SCADA→工业互联网平台 | 持续 | ★★☆ | 大企业数字化需求稳健，国产份额持续提升 |
| 数字孪生→工厂运维 | 早期 | ★★☆ | 用于产线模拟/故障预测/效率优化，成熟度在提升 |
| PLM/PDM→产品全生命周期管理 | 持续 | ★★☆ | 中高端被Siemens/PTC主导，国产替代空间大 |
| 工业安全→工控网络安全 | 早期 | ★☆☆ | 数字化程度提升→攻击面扩大，等保合规驱动 |

## 传导链

制造业劳动力成本↑ → 自动化+数字化投入↑ → 工业软件需求提升 → 出口管制→外购被限 → 国产工业软件进入"被迫使用"→"开始好用"的循环 → 标杆客户验证→行业复制→估值体系从"软件PE"→"平台PS"

## 周期定位

**工业软件处于从0→1向1→10过渡的关键期——政策+出口管制是催化剂，产品力提升是基础。** 当前产品力仍落后海外2-3代，但AI大模型的"能力平权"效应可能缩短差距。
