# MLCC/PCB/载板/CCL/连接器（P0 #8）

## 当前核心逻辑（主线）

AI服务器MLCC用量是普通服务器的3倍以上。英伟达Rubin ultra量产临近，服务器架构升级带动高容MLCC需求爆发。供需缺口从2025Q4开始拉大，进入涨价周期。

## 旧逻辑（正在弱化）

过去MLCC是消费电子周期驱动——手机、PC换机决定景气度。消费电子需求疲软时MLCC就过剩，靠汽车电子化托底。这个框架正在被AI算力改写。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| AI算力驱动→MLCC量价齐升 | 当前主线 | ★★★ | 最大的增量变化 |
| 汽车电子化→单车用量提升 | 持续 | ★★☆ | 稳定托底，不是弹性来源 |
| 消费电子复苏→补库周期 | 旧逻辑/边缘 | ★☆☆ | 曾经的主线，现在已非核心驱动力 |
| PTFE新材料→高速背板替代 | 新出现 | ★☆☆ | 验证中，落地会改写上游材料格局 |

## 传导链

AI资本开支↑ → 服务器出货结构向AI倾斜 → 高容MLCC需求×3+ → 产能利用率打满 → 缺货涨价 → 被动元件厂商利润弹性 → 上游PTFE等材料验证导入

## 周期定位

从0→1已完成（AI服务器已经开始拉货）。当前处于**1→10的爆发期**。
上一轮MLCC超级周期（2018年）由消费电子+汽车双轮驱动，本轮由AI单核引爆，结构更集中但弹性更大。

周期位置：**爆发中段**——缺货已确认但产能扩张尚未完成。

## 明星标的（需研报验证更新）

- MLCC龙头：村田/三星电机/TDK + 风华高科/三环集团
- 上游材料：PTFE、MLCC陶瓷粉相关
- PCB/载板：沪电股份/深南电路/兴森科技
- 连接器/线缆：安费诺/立讯精密
