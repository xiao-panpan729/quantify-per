# 显示面板（OLED/Mini/Micro/电子纸）— P0 #16

## 当前核心逻辑（主线）

**OLED从手机向IT/Monitor/TV的渗透加速，中国面板厂主导了本轮产能扩张。** 京东方/维信诺/深天马的OLED产能在2025-2027年集中释放，中国OLED产能全球占比从2023年的~30%提升到2026年的~50%+。IT OLED（平板/笔电）是最大增量——苹果iPad Pro已全面转向OLED，MacBook Pro OLED化在路上。

Micro LED是中长期方向（苹果暂停Micro LED手表项目说明技术路线仍有不确定性），Mini LED是过渡方案。电子纸在ESL（电子价签）领域年增速30%+，但市场空间远小于OLED。

## 旧逻辑（正在弱化）

面板是"强周期+重资产"——跟着TV面板价格周期走，投资节奏是"涨→扩产→跌→减产→涨"。京东方/华星光电被归类为"周期股"，估值无法突破。三星/乐金（LG）在OLED领域技术领先，中国面板厂在LCD领域做大规模。

现在：三星/LG退出LCD产能（转向OLED），中国面板厂在LCD领域寡头垄断（定价权增强），同时在OLED领域从"追赶"变为"并跑"。面板行业的属性从"周期股"变为"周期成长股"——中国龙头的底部利润和顶部利润同时抬升。

## 多重逻辑线

| 逻辑线 | 状态 | 强度 | 说明 |
|--------|------|------|------|
| OLED IT化→平板/笔电渗透 | 当前主线 | ★★★ | 苹果iPad/MacBook带动，2025-2027年产能翻倍 |
| LCD面板→中国寡头定价权 | 持续 | ★★★ | 京东方/华星光电+TCL，TV面板定价能力增强 |
| 车载显示→OLED取代LCD | 并行 | ★★☆ | 智能驾驶座舱多屏化，中控屏OLED渗透率提升 |
| Micro LED→下一代显示技术 | 早期 | ★★☆ | 苹果/Micro LED推迟说明技术仍不成熟 |
| Mini LED→iPad/MacBook过渡方案 | 过渡 | ★★☆ | 被OLED替代中（iPad Pro已转OLED） |
| 电子纸→ESL/电子书 | 并行 | ★☆☆ | 30%+增长但市场空间有限 |

## 传导链

AI端侧→换机周期启动 → 高端显示需求↑（OLED iPad/折叠屏/AR眼镜）→ 面板产线稼动率提升 → 中国面板厂新产线投产（B12/B15/B16等）→ OLED出货量翻倍 → 上游设备/材料受益（激光/蒸镀/掩膜版）

## 周期定位

**OLED处于从1→10的爆发中段。** 中国OLED产能从爬坡→满产的过程是2025-2027年最确定的放量逻辑。估值锚从"周期底部PB"向"成长PE"切换中。

LCD处于10→100的稳态期，周期波动仍在但振幅收窄——中国寡头格局形成后，价格战风险降低。

## 明星标的

- OLED面板：京东方 / 维信诺 / 深天马
- LCD面板：京东方 / TCL科技（华星光电）
- OLED设备/材料：精测电子（检测）/ 奥来德（蒸镀机）/ 莱特光电（发光材料）
- Mini/Micro LED：三安光电（外延芯片）/ 利亚德（LED显示）
- 电子纸：元太科技（E Ink寡头）/ 合力泰
- 掩膜版：路维光电 / 清溢光电
