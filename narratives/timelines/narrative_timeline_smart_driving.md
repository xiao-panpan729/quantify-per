# 智能驾驶叙事演化时间线

## 核心发现

**智能驾驶的叙事呈"爆品加速渗透"模式——一次发布会（比亚迪天神之眼）将高阶智驾从"30万以上高端选配"降维为"7万级标配"，L2+渗透率从8.5%到30%+的跃迁由此触发，叙事框架在1个月内完整建立。**

与半导体设备的"政策催化"和光模块的"速率跃迁"都不同，智能驾驶的投资逻辑核心是 **"成本突破×规模效应"**——垂直整合巨头决定"加量不加价"时，渗透率曲线斜率不可逆地改变。

这是所有AI链中**叙事建立最快、框架最集中**的一条——4篇标题命中文章全部集中在2025年2月~3月，2025-02-10比亚迪发布会后的两周内，汽车/AI/电子三团队联合覆盖，叙事框架一次建完，后续是渗透率数据的跟踪验证。

**分析师归属**：**程似骐（汽车首席）**主导整车及产业链估值重塑，陶亦然（联席）/陈怀山联席。于芳博（AI）提供端侧AI视角，刘双锋（电子）提供车载半导体/感知硬件视角。

---

## 时间线

### Phase 0：智驾早期——技术路线验证 + 高端选配定位（2024年11月~2025年1月）

**格局**：高阶智驾主要限制在30万以上高端车型，L2+渗透率仅8.5%。华为/小鹏/理想引领端到端技术路线验证，特斯拉FSD V13北美推送但尚未入华。

**关键状态**：
- 2024年L2+高阶智驾渗透率仅为8.5%
- 华为ADS 3.0（2024年8月上线端到端架构）
- 小鹏端到端大模型量产，理想"端到端+VLM"双系统
- 特斯拉FSD V13（2025年1月北美推送），规划2025年6月奥斯汀robotaxi商业运营
- 激光雷达仍在千元级，尚未降到"白菜价"

**叙事状态**：智驾是"高端选配"——有技术突破但规模受成本制约。智能化方向是共识，但拐点时间不确定。

---

### Phase 1：天神之眼——爆品引爆智驾平权（2025年2月）← 最关键拐点

**2025-02-10**：比亚迪智能化战略发布会——**全系标配高阶智驾**

**核心内容**：
- 「天神之眼」三版本矩阵：A版（三激光，仰望）、B版（激光，腾势/汉唐）、C版（三目5R12V，王朝海洋全系标配）
- **从6.98万海鸥到17.98万汉/唐，全部标配**
- **"加量不加价"**——价格体系与24款保持一致
- 城市记忆领航功能年底前OTA升级

**关键机制**（为什么是比亚迪引爆）：
> "智驾软件算法边际成本为零，硬件层的芯片等电子件采购成本对规模的敏感性远大于传统机械件，比亚迪通过垂直整合和规模化的优势，把高阶智驾的成本极致压缩。"——刘双锋（电子），2025-02-16周报

**2025-02-19/20**：程似骐团队报告《中高阶智驾标配超预期，看好整车及智驾产业链估值重塑》
- 这是中信建投体系内**智驾产业链估值重塑的第一个系统框架**

**2025-02-21**：**汽车+AI+电子三团队联合解读《"天神之眼"高阶智驾投资前景》**
- 汽车（程似骐）：整车估值重塑
- AI（于芳博）：端侧AI核心场景确认
- 电子（刘双锋）：车载半导体/感知硬件受益

**叙事升级**：从"高端选配"→ **"全民标配"** ，从"技术验证"→ **"渗透率跃迁"**

**叙事变化**：

| 之前（智驾=高端选配） | 之后（智驾=全民标配） |
|---------------------|---------------------|
| L2+渗透率8.5%，天花板明显 | L2+目标30%+，渗透率跃迁 |
| 智驾是增配选项 | 智驾是标配功能 |
| 只有30万以上车型搭载 | 7万级即可拥有 |
| 供应链验证中 | 零部件1-10阶段业绩兑现 |
| 算力方案分散 | SoC/域控技术迭代窗口打开 |

### Phase 2：产业链传导 + 端侧AI融合（2025年2月~3月）← 子主题展开

**2025-02-28**：电子团队AI终端系列——**高阶智驾加速普及**（刘双锋/何昱灵）

**核心内容**：
- 中芯国际上半年急单增多——半导体景气延伸至车载
- 比亚迪推动"智驾平权"，**上游三大核心环节**迎来发展机遇：
  1. **算力层**：车载SoC芯片与域控制器
  2. **感知层**：CIS图像传感器/光学镜头/激光雷达需求量增加
  3. **连接层**：高频高速连接器成为智能网联汽车标配

**2025-03-01**：**高阶智驾产业链投资机遇**——最完整的产业链全景图

**并行趋势**（AI端侧融合）：
- 包括比亚迪、吉利、长城等近20家车企宣布与DeepSeek深度融合
- 应用方向：智能座舱（语音交互/多模态）和AI运营（数据分析/营销）
- 意义：智驾硬件标配 + 座舱AI化，两条线并行

**子主题独立进度**：

| 时间 | 文章 | 子主题 |
|------|------|--------|
| 02-19/20 | 比亚迪中高阶智驾标配超预期 | **整车估值重塑**独立成篇 |
| 02-21 | 天神之眼高阶智驾投资前景 | **三团队联合**→汽车+AI+电子 |
| 02-28 | 高阶智驾加速普及 | **车载半导体/感知硬件**独立 |
| 03-01 | 高阶智驾产业链投资机遇 | **产业链全景**百科化完成 |

### Phase 3：端到端大模型迭代 + 全球对标（2025年3月~6月）

**端到端成为共识路线**：
- 华为ADS 3.0（2024-08上线）：端到端架构，2025年持续迭代
- 小鹏：端到端大模型量产，纯视觉方案降本
- 理想："端到端+VLM"双系统
- 特斯拉FSD V13：端到端神经网络，2025年1月北美推送

**FSD入华博弈**（影响估值的核心变量）：
- 规划2025年6月奥斯汀robotaxi商业运营
- FSD入华年内预期，但需中国监管批准
- 若FSD入华 = 技术对标锚确立，国内智驾估值体系有了全球坐标

**激光雷达降本加速**：
- 禾赛/速腾聚创合计市占率约83%
- 激光雷达降至千元级，推动中阶智驾下探

**叙事阶段**：从"国内爆品"→ **"全球技术路线对标"**

---

### Phase 4：当前——渗透率爬升 + 业绩兑现（2026年6月）

**最新状态**：
- 比亚迪天神之眼C已随新车型批量交付，城市记忆领航逐步OTA
- L2+渗透率从8.5%向30%+推进中
- 端到端大模型成为行业标准路线
- 激光雷达/车载芯片/连接器等上游环节持续放量

**未验证但重要的叙事锚点**：
- FSD入华进展（受监管不确定性影响最大）
- robotaxi商业模式在海外验证情况
- L2+渗透率能否如期达30%+

| # | 时间 | 叙事升级 | 关键事件 | 本质 |

**独特之处**：这是所有AI链中**叙事建立速度最快的**——从发布会到完整产业链框架仅用3周，之后几乎没有大的叙事切换，进入渗透率跟踪验证阶段。

## 当前叙事定位

智能驾驶处于 **"爆品引爆后渗透率验证"阶段**——比亚迪天神之眼已证明"智驾可以很便宜"，框架已建立完毕，后续核心是跟踪L2+渗透率能否从8.5%拉到30%+，以及FSD入华能否打开新的技术对标空间。

**最新核心逻辑（2026-06）**：比亚迪天神之眼"加量不加价"引爆智驾平权 → 全系标配推动L2+渗透率从8.5%向30%+跃迁 → 上游三大环节（SoC域控/感知硬件/高速连接器）进入1-10放量阶段 → 端到端大模型成为行业共识 → FSD入华是下一个重要催化剂。

**注意**：由于叙事框架在2025年2月已完整建立，后续中信建投的智驾覆盖进入"跟踪验证"而非"框架重塑"阶段，文章密度下降是自然现象——不是主题降温，而是框架已固化。
