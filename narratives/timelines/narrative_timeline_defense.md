# 国防军工叙事演化时间线

## 核心发现

**国防军工的叙事呈"三轮驱动"模式——黎韬扬团队构建的"内需筑基、外贸扩张、民用反哺"框架（第一曲线国内基本盘/第二曲线军贸新引擎/第三曲线军技民用新边界），将军工从"周期成长"重新定位为"全面成长"，叙事从订单博弈升级为产业趋势判断。**

与商业航天的"双轨共振"（政策轨+产业轨交替催化）不同，国防军工的三轮驱动是**嵌套结构**——内需是底盘，军贸是弹性放大器，军转民是长期空间打开器。三曲线的成熟度和弹性决定了投资优先级：军贸（第二曲线）弹性最大，内需（第一曲线）确定性最高，军转民（第三曲线）时间最长。

**分析师归属**：**黎韬扬（军工与新材料首席）**全程主导，团队覆盖全链。任宏道（联席首席+建材首席，北航本硕/7年航天院所总体设计经验）提供产业深度，王春阳（清华MBA+交大船舶）提供舰船/海军视角。阎贵成/武超则（通信团队）在2025年4月跨界覆盖国防信息化/军工AI。

## 时间线

### Phase 0：板块筑底期——结构性复苏前夜（2024年12月~2025年3月）

**格局**：军工板块自2023年起进入调整期，受军品采购低成本、人事调整等多重因素影响，整体业绩持续承压。除船舶受益民船大周期外，航空/航发/信息化等板块均处于业绩底部。

**2025-03-18**：**军工年报呈现结构性特点，建议关注新域新质方向**（黎韬扬）

**关键判断**：
- 板块业绩接近底部，2025年有望触底回升
- 核心公司发布合同公告，关联交易额预计增加
- **新域新质方向**（低成本精确制导弹药/无人系统/智能化作战底座）表现明显优于传统板块
- 中证军工PE 71.86倍，估值历史中位

**叙事阶段定位**：
> "军工行业已逐步进入新一轮周期，由'量价齐升'过渡到'量增价稳'阶段，由'平台放量'过渡到'建体系，补短板'阶段，由'全面增长'过渡到'结构性增长'阶段。"——黎韬扬，2025-03-18

### Phase 1：框架建立——信息化+AI跨界覆盖 + 三大增长曲线（2025年4月~6月）← 框架拐点

**2025-04-09**：**国防信息化与智能化建设有望提速**（阎贵成/武超则/刘永旭——通信团队跨界）

**意义**：通信团队首次系统覆盖军工AI赛道，将光通信/卫星互联网/数据链等与军工信息化融合。

**核心内容**：
- AI正深度重构军事作战模式（OODA循环加速）
- 美军Palantir市值已超2000亿美元，远超越波音等传统军工巨头
- 我军在无人机蜂群/机器狼集群实现突破
- 卫星互联网加速建设（GW星座+千帆星座）
- 相控阵雷达渗透率持续提升

**2025-06-13**：国防军工需求有望回暖（黎韬扬）
- 乌克兰无人机袭击俄空军基地，体系化/智能化重新定义作战模式

**2025-06-16**：**2025下半年投资策略——"建体系，补短板，走出去"← 三大增长曲线框架首次建立**

**核心框架**（奠定后续12个月分析基础）：

| 增长曲线 | 定位 | 驱动因素 | 成熟度 |
|---------|------|---------|-------|
| **第一曲线**国内军工需求 | 基本盘 | 国防预算稳定增长+装备升级换代 | ★★★ 确定性最高 |
| **第二曲线**军贸需求 | 新引擎 | 印巴冲突催化+实战验证+性价比优势 | ★★★ 弹性最大 |
| **第三曲线**泛军工需求（大飞机/低空经济/商业航天/深海科技） | 新边界 | 军技民用，万亿级市场 | ★★☆ 长期空间 |

**叙事升级**：从"板块筑底/结构性复苏"→ **"三大增长曲线框架"**

### Phase 2：业绩拐点 + 军贸DeepSeek时刻（2025年9月~11月）← 验证拐点

**2025-09-03**：**抗战胜利80周年阅兵**——无人集群/高超音速导弹等新型装备展示
- 市场对装备迭代进程与列装规模的关注升温
- 阅兵前Q2-Q3板块冲高，阅兵后市场逻辑从"事件预期"→"业绩验证"

**2025-11-06**：**三季报综述——板块业绩拐点已现**（黎韬扬）

**关键数据**：
- 2025Q3营收6,003.75亿元，同比+16.99%
- 归母净利润298.22亿元，同比+14.01%
- **国防信息化（+73.60%）、航空（+26.51%）、航天弹（+26.14%）**增速亮眼
- 下游整机（+48.92%）与中游分系统（+89.71%）利润增长强劲
- 上游材料（-14.36%）仍承压，产业链议价能力分化明显

**2025-11-11**：**2026年投资策略展望——"建体系，走出去，军转民"**（黎韬扬）
- 框架升级："补短板"→**"军转民"**，第三曲线地位提升

**关键论点——军贸迎来"DeepSeek时刻"**：
> "中国军贸从'追随者'蜕变为'规则制定者'。技术层面，歼-10CE战机实战验证打破'低端廉价'刻板认知；服务模式，中国提供全生命周期支持（巴基斯坦可自产枭龙85%零件）；战略层面，中国将军贸嵌入'一带一路'，形成'安全-发展'双循环。"——黎韬扬，2025-11-11

**叙事变化**：

| 之前（2025-06中期策略） | 之后（2025-11年度策略） |
|----------------------|----------------------|
| "建体系，补短板，走出去" | **"建体系，走出去，军转民"** |
| 军贸=新兴方向 | 军贸=**DeepSeek时刻（体系化出口）** |
| 第三曲线=泛军工需求 | 第三曲线=**军转民（六大方向）** |
| 行业=周期成长 | 行业=**全面成长**（三轮驱动） |

### Phase 3：年报验证 + 国防预算 + 无人系统爆发（2026年1月~4月）

**2026-03-11**：**国防预算7%增长，关注军转民投资机会**（黎韬扬）
- 2026年国防支出19,095.61亿元，同比+7%（2025年7.2%小幅下调，但绝对额持续增加）
- **无人系统业绩大幅改善**：中无人机营收+340.11%，归母+264.28%
- 增材制造板块：铂力特+40.54%，华曙高科+45.43%
- 预计十五五期间国防预算保持7-7.5%稳定增长

**2026-04-09**：**军工板块业绩分化**（黎韬扬）
- 上游电子元器件：火炬电子/鸿远电子净利润增速均60%以上
- 船舶：净利润同比+167.26%（民船大周期）
- 航发：航发控制净利润-55.43%，持续承压
- 无人系统：基本面已触底回升

**叙事阶段**：从"业绩拐点"→ **"结构性复苏确认"**

### Phase 4：持仓回暖 + 十五五预期（2026年5月~6月）

**2026-05-19**：**2026Q1基金军工持仓结构性复苏**（黎韬扬）
- 军工持仓占比3.97%，较25Q1低点（3.51%）回升0.46个百分点
- **持仓呈现"先探底、后波动回暖"特征**
- 核心博弈点已从"十四五末期事件驱动"转向**"十五五规模化订单下达节奏"**

**2026-05-20**：**2025年报&2026Q1综述——板块业绩触底回升**（黎韬扬）

**关键数据**：
- 2025全年：营收8,314.93亿元（+8.44%），归母318.90亿元（+18.50%）
- **2026Q1：营收1,840.58亿元（+19.36%），归母107.12亿元（+47.24%）**
- 利润率小幅回暖（毛利率17.91%→18.96%）
- 26Q1经营性现金流5年来首次回正

**最新框架总结**：
> "中国军工产业已从过去依赖国内单一需求的模式，演进为三轮驱动的新发展格局——'内需筑基、外贸扩张、民用反哺'正在深刻重塑我国军工产业的面貌和边界。行业从'周期成长'转向'全面成长'。"——黎韬扬，2026-05-20

## 当前叙事定位

国防军工处于 **"业绩触底回升+十五五预期建立"的叠加期**——2025年报已验证拐点，2026Q1归母+47.24%确认加速。核心叙事焦点正在从"业绩拐点确认"向"十五五规模化订单下达"切换，军贸（第二曲线）提供弹性，军转民（第三曲线）打开长期空间。

**最新核心逻辑（2026-06）**：国防预算19095.61亿元，稳定增长7% → 十四五收官订单集中释放，2026Q1归母+47.24% → 十五五规模化订单成为下一核心催化 → 军贸DeepSeek时刻（歼-10CE实战验证+体系化出口）提供弹性 → 无人系统/增材制造新域新质方向业绩爆发 → 军转民（商业航天/低空经济/大飞机）打开长期边界

**注意风险**：估值偏高（中证军工PE 118倍），航发/国防信息化等板块利润仍承压，十五五订单节奏不确定。

**叙事升级**：从"业绩触底回升"→ **"十五五规模化预期+全面成长格局"**
