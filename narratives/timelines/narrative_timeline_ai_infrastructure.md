# AI基础设施叙事演化时间线

## 核心发现

**AI基础设施是中信建投覆盖密度最高的产业链（113篇标题命中/763篇内容提及），叙事演变呈"分形扩散"特征**：

从2025-07的500页AI新纪元巨著建立统一框架开始，AI基础设施的叙事像一棵树一样不断分叉——每3-4个月，一个子主题就独立成新的叙事分支。与功率半导体的"单点爆发"和先进封装的"碎片拼凑"都不同，这是**"大树分蘖"模式**。

---

## 时间线

### Phase 0：分散覆盖期（2025年3月~6月）

**格局**：AI基础设施的各个子板块（算力、液冷、IDC、光模块）分散在不同分析师团队的报告里，尚无统一框架。

| 时间 | 事件 | 叙事状态 |
|------|------|---------|
| 2025-03~06 | 通信团队讲光模块/CPO | 算力=网络连接 |
| 2025-06-17 | 电子团队讲端侧AI爆发 | 算力=GPU芯片+封装 |
| 2025-06-16 | AI团队讲推理走向舞台中央 | 算力=AI芯片+自主可控 |

三个团队各讲各的，没有一个"算力基础设施"的统一叙事。

---

### Phase 1：框架锚定（2025年7月）← 最关键拐点

**引爆点**：2025-07-25 中信建投发布500页/40万字深度研报《AI新纪元：砥砺开疆・智火燎原》

这是中信建投历史上最宏大的单篇报告之一，20多位分析师联合撰写，从AI模型→算力基础设施→应用场景全链条覆盖。

**核心贡献**：将"算力基础设施"确立为一个统一的投资主题

**关键论点**：
- AI算力消耗从训练转向推理，带来显著增量
- AI对产业的渗透速度超过互联网革命
- 算力基础设施 = GPU/ASIC + 液冷 + 铜连接 + PCB + 光模块 + 电源 + 先进封装
- 首次将液冷从"概念"提升为"刚需"
- 首次提出中国供应商在液冷/铜连接领域的机会

**叙事演变（从Phase 0→1）**：

| 之前（分散覆盖） | 之后（统一框架） |
|----------------|----------------|
| 算力=各自团队的子板块 | 算力=统一的投资主题 |
| 液冷=IDC的一个配件 | 液冷=算力基础设施的核心创新方向 |
| 铜连接=通信技术细节 | 铜连接=超节点架构的关键瓶颈 |
| 国产算力=替代故事 | 国产算力=2025年发展大年 |

**关键变量（首次系统化提出）**：
- 英伟达芯片功耗演进（700W→1200W→2000W+）
- 液冷渗透率（2025年20%→2027年50%+）
- 铜连接市场空间（NVL72单机柜约$4万价值量）
- CSP资本开支增速
- 国产芯片份额变化

**节点地图对照**：芯片板块 2025-06-17~2025-09-08 节点（+30.34%, 60d, Q=11/A）。这一波行情正好对应从"分散覆盖"→"统一框架"的叙事建立期。

---

### Phase 2：子主题独立化（2025年8月~10月）

500页巨著之后，各子主题开始从"算力基础设施"大树中分蘖，形成独立的跟踪线：

**子主题独立时间线**：

| 日期 | 文章 | 独立的子主题 |
|------|------|-------------|
| 08-14 | AI算力产业链投资前景 | 算力芯片产业链整体框架 |
| 09-06 | 液冷散热系列报告一：热界面材料 | **液冷首次独立成系列** |
| 10-11 | 算力芯片产业链投资机遇（135KB） | GPU/ASIC/国产芯片深度 |
| 10-28 | 液冷散热板块投资机遇 | 液冷从"系列报告"→"独立板块" |

**叙事特征**：这个阶段的核心特征是**"主题扩散"**——500页巨著划了一个大圈，然后每个子主题都开始在这个圈里长出独立的根。

### Phase 3：年度展望 + 产业革命定性（2025年11月~12月）← 第二次叙事升级

**2025-11-26**：算力产业链2026年投资展望

**叙事升级**：从"算力需求增长"→ **"AI产业革命类比工业革命"**

关键表述：
> "目前世界正处于AI产业革命中，类比工业革命，影响深远，不能简单对比近几年的云计算、新能源等，需要以更长期的视角、更高的视野去观察。"

**核心催化**：OpenAI 3000亿美元算力采购协议（与Oracle）+ 阿里3800亿AI基建计划

**叙事变化**：

| 维度 | Phase 2（算力需求增长） | Phase 3（AI产业革命） |
|------|------------------------|---------------------|
| 时间视野 | 1-2年 | 10年+ |
| 增长驱动 | 芯片迭代 | Token指数级增长+Agent普及 |
| 竞争格局 | 英伟达主导 | 多路线并行（GPU/ASIC/国产） |
| 瓶颈环节 | 芯片供给 | 电力+散热+网络带宽 |
| 国内叙事 | 国产替代 | Local for Local 产业重构 |

**子主题继续分蘖**：
- 11-24: AIDC配储（北美缺电叙事）
- 12-18: 液冷散热板块投资前景（液冷完全独立）
- 12-11: PCB产业链2026年展望（算力→PCB链路打通）

### Phase 4：缺电叙事+AIDC电源革命（2026年1月~3月）← 第三次叙事升级

**2026-01-15**：AIDC电源革命正式开启

**叙事升级**：从"算力基础设施"→ **"电力即算力"**

这个阶段的核心叙事变化：**算力基础设施的投资逻辑从"买芯片"延伸到"买电"。**

缺电叙事给算力基础设施添加了"稀缺性溢价"——数据中心不再只是一个IT成本项，而是变成了电力基础设施的一部分。

**子主题独立进度**：

| 日期 | 文章 | 子主题 |
|------|------|--------|
| 01-15 | AIDC电源革命（四大方向） | 电源/储能/功率半导体 |
| 01-28 | AIDC电源产业链投资前景 | 电源产业链 |
| 02-28 | AIDC产业链投资前景 | AIDC全产业链（含燃气轮机等） |
| 03-07 | 缺电产业链投资机遇 | **"缺电"成为独立主线** |
| 03-13 | 液冷一次侧设备迎来价值重估 | 液冷设备价值量分析 |
| 03-21 | 持续看好缺电产业链 | 缺电叙事被市场接受 |
| 03-28 | AI算力产业链投资机遇 | 算力链最新全景 |

## 当前叙事定位

AI基础设施处于 **"持续跟踪+不断分蘖+业绩验证"** 的稳态。与功率半导体等待验证不同，AI基础设施的叙事引擎（Token指数级增长/大模型商业闭环/Agent普及）仍在持续运转，2026年H1多项关键数据验证了需求的真实性和持续性，已从"主题投资"进入"业绩验证期"。

**最新核心逻辑（2026-06）**：我们很可能在2026年看到ARR达到2000亿美元左右的大模型公司 → 算力需求没有天花板 → 2026年H1多项高频数据验证需求真实性(NVDA营收+85%，CSP CAPEX 7100亿，台积电+40.6%) → 从GPU→光模块→液冷→电源→光纤整条链持续受益
