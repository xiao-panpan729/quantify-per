# 光互联叙事演化时间线

## 核心发现

**光互联的叙事呈"速率跃迁"模式——每代速率升级（800G→1.6T→3.2T）重置投资主题，CPO作为颠覆性技术叠加，形成"渐进升级+范式切换"的双层叙事结构。**

与AI基础设施的"大树分蘖"（一个框架不断分叉）不同，光互联的叙事演进有明确的**技术路线图**——英伟达GPU平台从Blackwell→Rubin→Feynman，对应光模块从800G→1.6T→3.2T，升级周期压缩到1年一代，叙事节奏精确到季度。

**与中国大多数AI产业链的"追赶逻辑"根本不同，光模块是中国AI产业链中少数已全球领先的环节**——中际旭创/新易盛/天孚通信合计全球份额>50%，壁垒已建立。

**分析师归属**：阎贵成（通信首席）团队主导，杨伟松（科技前瞻，6年Coherent光通信产业背景）提供技术深度，刘永旭联席。

## 时间线

### Phase 0：通信周期股定位（2025年3月~7月）

**格局**：光模块主要被视为"通信板块"的子行业，叙事围绕CSP资本开支周期展开。

| 时间 | 事件 | 叙事状态 |
|------|------|---------|
| 2025-03-25 | NVIDIA发布Blackwell Ultra + 三款CPO交换机 | CPO从"实验室技术"→"商用导入窗口打开" |
| 2025-04-08 | OFC聚焦光子新技术（CPO/硅光/薄膜铌酸锂） | 技术路线百花齐放，但尚未形成投资主题 |
| 2025-06-14 | 通信行业2025中期策略：稳居红利，精选算力 | 光模块=算力板块的一部分，尚未独立成篇 |

**叙事定位**：这个阶段光模块还处于"通信周期股"的旧框架——市场把它当作跟随云厂商资本开支的周期性板块，而不是AI结构性增长的核心受益者。400G→800G的升级被认为是"渐进式"的，不像后来的叙事那样"变革性"。

**关键信号**：2025年GTC大会上英伟达发布三款CPO交换机（Quantum 3450-LD/Spectrum SN6810/SN6800），但市场尚未充分理解其意义。

### Phase 1：AI新纪元锚定 + 光模块首次独立叙事（2025年7月~8月）← 最关键拐点

**2025-07-25**：中信建投发布500页深度研报《AI新纪元》，首次将光模块纳入"算力基础设施"统一框架。

**2025-08-16**：**光模块赛道投资机遇**——这是光模块在中信建投体系内**首次以独立赛道身份**出现。

**叙事升级**：从"通信周期股"→ **"AI算力基础设施核心环节"**

**关键内容（2025-08-16）**：
- 800G光模块2023年开始放量，2024-2026年保持高速增长
- 1.6T光模块2025年开始出货，2026年有望放量
- 国内光模块巨头占据全球主要份额
- 硅光与CPO是两大技术趋势
- 公募通信持仓创历史新高（占比3.90%），重点加仓光模块/光器件

**叙事变化**：

| 之前（通信周期股） | 之后（AI算力核心） |
|----------------|----------------|
| 跟随CSP资本开支周期 | 跟随AI算力需求指数增长 |
| 400G→800G是渐进升级 | 800G→1.6T是革命性跃迁 |
| 周期性估值（15-20x PE） | 成长性估值（30-40x PE） |
| 国内厂商=追赶者 | 国内厂商=全球领导者 |

### Phase 2：业绩验证 + Scale-up叙事建立（2025年9月~12月）

**2025-12-23**：**光模块2026年投资机遇**——尺度从"产品升级"扩展到"架构变革"

**数据验证**：
- 2025Q1-Q3光模块/光器件板块营收662亿元，同比+65%
- 归母净利润169.4亿元，同比+123%
- 在所有通信子板块中表现最好

**Scale-up叙事**（最重要的新逻辑）：
- 传统理解：GPU:光模块=1:2.5（Scale-out网络）
- Scale-up新空间：若采用fat-tree架构，GPU:800G光模块=1:36
- 博通CEO：Scale-up市场空间是Scale-out的5~10倍
- 若未来Scale-up全部采用光模块，市场空间是现在的5-8倍

**英伟达Roadmap（叙事锚点）**：
- Blackwell（2023-24）：1800GB/s NVLink 5，800G CX8
- Rubin（2025-26）：3600GB/s NVLink 6/7，1600G CX9，Spectrum6 CPO
- Feynman（2027-28）：NVLink 8，204T Spectrum7 CPO

**叙事升级**：从"1.6T放量"→ **"Scale-up打开5-8倍增量空间"**

### Phase 3：CPO从概念到产业订单（2026年1月~4月）← 第二次叙事升级

**2026-02-10**：**北美CSP资本开支强劲增长，继续推荐CPO、光纤行业**

**关键事件**：Lumentum获CPO相关数亿美元订单，预计2026Q4 CPO营收约5000万美元，**2027年上半年将迎来爆发**

**叙事意义**：CPO从"实验室技术"→ **"有订单、有营收、有爆发时间表"**

**2026-03-17**：关注OFC与GTC大会投资机会
- Arista牵头组建XPO MSA，定义12.8Tbps液冷光学模块
- 光纤光缆"供给偏紧、量价齐升"

**叙事变化**：

| 之前（CPO=远期概念） | 之后（CPO=近期产业） |
|-------------------|-------------------|
| "2027-2028年可能商用" | "2026Q4有营收，2027H1爆发" |
| 科技巨头在实验室验证 | Lumentum拿到数亿美元订单 |
| 交换机侧应用为主 | IO领域突破电信号速率瓶颈 |
| GP:光模块=1:2.5 | GP:光引擎=1:11.5（Scale-up CPO） |

---

### Phase 4：上游紧缺 + 多技术路线 + 3.2T布局（2026年5月~6月）

**2026-05-16**：**光模块产业投资机遇**——最新全景

**上游供应紧缺成为叙事新焦点**：
- **光芯片（EML/CW激光器）**：InP衬底/生产设备受限，供不应求
- **光隔离器**：法拉第旋光片产能紧张，日本住友/美国Coherent主导，国产替代加速
- **MPO连接器**：多芯光纤连接器需求高速增长，CPO/OIO将进一步推高用量

**谷歌TPU V8发布**（2026年4月Cloud Next）：
- V8t/V8i ICI带宽提升至2400GB/s（前代1200GB/s）
- 光模块从800G→**1.6T升级**
- Boardfly新拓扑（应对推理需求）

**3.2T布局+多技术路线**：
- 薄膜铌酸锂（TFLN）：未来单通道400G+最优方案
- Slow and wide：英伟达方案，数百低速并行通道替代少量高速通道
- Fast and wide：200G单通道，可插拔和CPO/NPO广泛应用

**2026-06-04**：《AI产业全景图谱》新书节选——内部互连：铜连接的性价比复兴（百科化完成）

### Phase 5：光纤光缆超级周期 + CPO全面量产（2026年6月—）

**格局**：光纤光缆行业迎来十年一遇的超级周期——G.652D光纤价格涨幅超400%，出口量价齐升。同时英伟达Spectrum-X CPO交换机全面量产，光互联从"概念"彻底走向"产业落地"。

**2026-06-XX**：**G.652D光纤价格暴涨400%，光缆供给全面紧缺**
- G.652D光纤价格从25元涨至80元/芯公里，涨幅超400%，创历史新高
- 光纤出口量同比增长63.6%，出口金额同比增长126.8%，量价齐升格局确认
- 中国月产量的65%用于出口，全球供给高度依赖中国产能
- **叙事阶段**：光纤光缆从"通信周期品"升级为"AI算力基建核心材料"——算力集群互联需求拉动光纤进入超级周期

**2026-06-XX**：**英伟达Spectrum-X CPO交换机全面量产**
- 英伟达Spectrum-X CPO交换机全面进入量产阶段，CPO时代正式开启
- GPU与光引擎配比达1:11.5（Scale-up场景），光引擎需求呈指数增长
- CPO交换机量产推动1.6T光模块加速放量，3.2T布局同步推进
- **叙事阶段**：CPO从"订单验证"→ **"全面量产"**——CPO产业链进入业绩兑现阶段

**2026-06-XX**：**800G需求持续强劲，1.6T快速放量**
- 800G光模块需求维持高景气，1.6T光模块进入快速放量通道
- 源杰科技股价超越茅台成为A股最新高价股，光芯片龙头估值重塑
- 多技术路线（硅光/TFLN/slow-wide）同步推进，行业进入代际切换密集期
- **叙事阶段**：光模块从"1.6T预期"→ **"1.6T业绩兑现"**——估值从PE驱动切换为EPS驱动

## 与其他链的对比

| 维度 | AI基础设施(#11) | AI大模型(#9) | 光模块(#12) |
|------|----------------|-------------|------------|
| 叙事模式 | **大树分蘖**：一个框架→多个子主题 | **三浪叠加**：模型/Agent/多模态并行 | **速率跃迁**：速率升级重置投资主题 |
| 核心驱动 | Token指数级增长 | 模型迭代（技术推动） | 带宽需求（AI集群互联） |
| 升级节奏 | 持续深化 | 3个月一轮模型迭代 | 1年一代速率升级 |
| 中国定位 | 追赶/替代 | 部分领先 | **全球领先**（份额>50%） |
| 业绩验证 | 资本开支（先行指标） | ARR/Token（后验指标） | **财报直接验证**（营收/出货量） |
| 颠覆性威胁 | 低（需求没有天花板） | 中（模型竞争激烈） | **高**（CPO可能颠覆可插拔格局） |

**关键区别**：光模块是所有AI链中**唯一中国已全球领先**的环节。这意味着它的叙事逻辑不是"替代/追赶"而是"份额维持+代际升级"——受益逻辑更确定，但也面临CPO技术路线的颠覆风险。

## 当前叙事定位

光模块处于 **"业绩兑现+技术跃迁"的叠加期**——800G持续放量有业绩支撑，1.6T开始放量打开估值空间，CPO从概念走向产业提供期权价值，上游供应紧缺创造新投资机会。

**最新核心逻辑（2026-06）**：AI集群带宽需求没有天花板 → 800G维持高增，1.6T放量，3.2T开始布局 → Scale-up打开5-8倍增量空间 → CPO从概念→订单→营收，2027年爆发 → 上游光芯片/隔离器/MPO紧缺 → 多技术路线（硅光/TFLN/slow-wide）百花齐放

**注意风险**：CPO作为颠覆性技术，若大规模商用可能重塑可插拔光模块的格局。但光模块公司凭借光引擎设计制造能力，在CPO时代仍具备核心优势。

**2026年6月新增维度**：
- 光纤光缆超级周期：G.652D价格暴涨超400%，出口量+63.6%/金额+126.8%，月产量65%出口，全球供给高度依赖中国
- CPO全面量产：英伟达Spectrum-X CPO交换机量产，GPU:光引擎=1:11.5（Scale-up场景），CPO进入业绩兑现阶段
- 源杰科技超越茅台成为A股最新高价股，光芯片龙头打开板块估值天花板
