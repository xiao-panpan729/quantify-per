# 功率半导体叙事演化时间线

## 核心发现

**功率半导体在2025年11月~2026年1月间完成了一次根本性的叙事迁移**：从"跟着新能源车/光伏走的IGBT周期股"→"AI数据中心电源革命的核心瓶颈（SiC/GaN）"。

这次叙事拐点可以精确到 **2026年1月15日**——中信建投发布《AIDC电源革命正式开启》当天。

---

## 时间线

### Phase 1：旧叙事末期（2025年7月~10月）

**主导逻辑**：功率半导体 ≈ IGBT ≈ 新能源车 + 光伏逆变器

当时市场对功率半导体的认知是：
- IGBT跟着新能源车销量走，周期性明显
- SiC是"未来技术"，还在等上车验证
- 功率半导体讨论出现在"电子团队研究成果合集"中作为子板块之一
- AIDC作为一个主题已经出现，但讲的是储能/光伏/锂电，**功率半导体完全没有出现在AIDC叙事中**

| 日期 | 文章 | 隐含叙事 |
|------|------|---------|
| 08-18 | AIDC高景气延续，关注锂电、光伏反内卷进展 | AIDC = 锂电+光伏的事 |
| 09-01 | AIDC、储能等高景气延续，机器人、氢能长期潜力 | AIDC = 储能的事 |
| 09-24 | 电子团队研究成果合集 | 功率半导体=电子子板块 |
| 10-28 | 液冷散热板块投资机遇 | AIDC散热=液冷的事 |

**关键变量（当时市场关注）**：
- 新能源车月销量
- IG BT 交货周期（是否缓解）
- SiC 上车进度（蔚来/比亚迪等车型）

**节点地图对照**：功率半导体板块 2025-06-17~2025-09-05 节点（+32.2%, 59d）。这一波走的仍然是新能源车+光伏的传统逻辑。

---

### Phase 2：AIDC认知铺路（2025年11月）

**转折信号**：AIDC配储成为主题，但功率半导体仍未进入视野

2025年11月24日，中信建投发布《再论AIDC配储是解决北美电容量缺口的必然选择》。这是第一篇深度聚焦AIDC供电架构的报告。

**核心内容**：
- 重申800V HVDC是趋势
- 测算2026-2028年AI电力需求32-71GW
- AIDC配储是必选项（不是可选项）
- 讨论OBBB法案对中国储能的影响

**关键发现**：这篇11000字的报告中，**SiC/GaN/功率半导体几乎未被提及**。叙事焦点在"电从哪里来"（储能/SOFC电池），而非"电怎么变"（功率半导体）。

**节点地图对照**：功率半导体板块 2025-11-25~2026-02-04 节点（+30.02%, 50d）。**这一波启动时市场的叙事框架仍然是"新能源车+光伏"**，但在运行过程中，叙事框架完成了切换。

| 时间点 | 驱动叙事 | 事后看 |
|--------|---------|--------|
| 11-25（节点起点） | 新能源车年末冲量+光伏装机旺季 | 旧逻辑尾声 |
| 12月~1月初 | 市场开始注意到AIDC电源主题，但功率半导体不突出 | 叙事蓄力 |
| 01-15（报告发布） | AIDC电源革命正式提出功率半导体为核心环节 | **叙事正式切换** |
| 02-04（节点终点） | 春节前行情收尾 | 旧节点完结 |

---

### Phase 3：叙事正式切换（2026年1月15日~17日）

**爆发点**：中信建投发布重磅深度报告（1.5万字+），将功率半导体推到AIDC电源革命的核心位置

**三道冲击波**，三天内密集发布：

| 日期 | 报告标题 | 内容 |
|------|---------|------|
| 01-15 | AIDC电源革命正式开启，电源主机、储能、功率半导体、核心部件四大方向共振 | **全面框架**：SST技术结构+投资标的+千亿市场测算 |
| 01-16 | AIDC电源产业投资建议 | **选股逻辑**：海外链优先，电源主机/配储/零部件/三代半 |
| 01-17 | 各厂商AI芯片向大功率不断迭代，推升单机柜功率迅速增长 | **需求论证**：英伟达/谷歌芯片功率持续提升趋势 |

**核心论点（叙事新框架）**：

1. **SiC/GaN从"未来技术"变成"现在必须"**
   - SST中的功率半导体成本占比30-40%
   - 800V HVDC架构必须用SiC MOSFET（耐压10-15kV）
   - GaN是48V/50V DC/DC降12V的唯一高效方案
   - 纳微半导体已出10kW 800V→50V参考设计，效率98%

2. **传导路径重新定义**
   - 旧：新能源车→IGBT→跟着整车周期走
   - 新：AI算力→芯片功耗提升→800V HVDC→SST替代变压器→SiC/GaN需求爆发

3. **市场空间重估**
   - 旧：功率半导体是个跟随市场（跟着车/光/工控走）
   - 新：仅AIDC电源SiC器件需求就将达百亿级，SST系统千亿级

**关键变量（新框架下的先行指标）**：
- 英伟达芯片功率（B200:1000W→Rubin Ultra:900W+/芯片，NVL72→Kyber MW级）
- SST样机交付进度（台达1.2MW样机、西电2.4MW试点）
- 800V HVDC标准落地（英伟达推800V vs OCP推±400V，实际看Rubin Ultra量产节奏）
- 云厂商资本开支（影响2027~2028年放量节奏）
- SiC衬底产能利用率（决定半导体器件环节供需格局）

### Phase 4：叙事扩散与扩展（2026年1月~4月）

**叙事从"AIDC电源革命"扩展为"缺电产业链"**

| 日期 | 文章 | 叙事演变 |
|------|------|---------|
| 01-28 | AIDC电源产业链投资前景 | 框架固化 |
| 02-15 | 图说AIDC电源革命正式开启 | 视觉化传播 |
| 02-28 | AIDC产业链投资前景 | 范围扩大（含芯片/光模块） |
| 03-07 | 缺电产业链投资机遇 | **叙事升级**：从"电源革命"→"缺电危机" |
| 03-14 | 持续看好储能产业链 | 重心回储能 |
| 03-21 | 持续看好缺电产业链投资机遇 | 缺电叙事被市场接受 |
| 04-05 | 储能&锂电产业链投资机遇（IGBT匹配） | 功率半导体回到储能链 |
| 04-23 | 半导体设备产业链投资机遇 | 扩散到设备 |

**关键观察**：到3月，叙事从"AIDC电源革命"升级为"缺电产业链"——这是一个更大的框架，功率半导体从主角之一变成缺电解决方案的零部件环节。叙事在扩展中有所稀释。

**节点地图对照**：功率半导体板块 2026-04-02~2026-05-26 节点（+37.29%, 35d）。这一波比前一波幅度更大但时间更短，说明市场对新叙事的接受度更高但兑现预期加速。

### Phase 5：当前状态（2026年5月~6月）

- 功率半导体叙事从"爆发"转为"消化验证"
- 关键变量兑现进度：SST仍处于试点→小批量阶段，Rubin Ultra量产节点尚未到来
- 中信建投5月后缺少新的功率半导体深度报告，叙事处于等待验证期
- 板块进入高位震荡，新旧逻辑并存
