# MLCC/PCB/载板叙事演化时间线

## 核心发现

**MLCC/PCB的叙事呈"AI驱动升级+超级周期"的双轨模式——PCB是一条自2025年8月起缓慢爬升的"AI价值升级链"（从板厂到CCL到上游树脂/玻纤布/铜箔逐级传导），MLCC则是一条在2026年5月底集中爆发的"超级周期"链（AI服务器+电动车双重需求引爆缺货潮），两条子链在不同时间以不同节奏展开但最终在2026年6月汇合为"算力需求引发的电子元器件全面景气"。**

PCB的叙事核心是**价值量升级**而非简单的量增——AI服务器架构演化（从H100到GB200到Rubin）驱动PCB层数从12层向22层+升级、CCL材料从M6向M9升级，单台PCB价值量翻倍以上。这一升级效应从PCB板厂→CCL→上游树脂/玻纤布/铜箔逐级传导，形成一条完整的"AI材料升级链"。

MLCC的叙事核心是**超级周期**——MLCC作为"工业大米"，需求与电子周期深度绑定（15年周期律）。AI服务器MLCC用量是传统服务器的2-3倍，电动车MLCC是燃油车的6倍，双重需求同时爆发导致高端MLCC供不应求、产能利用率突破90%、涨价通道开启。

**分析师归属**：双轨各自独立——**PCB线**由**许光坦（机械）**率先覆盖（设备视角），**刘双锋（电子）**提供云厂资本开支+PCB扩产视角，**于芳博（AI）**提供全球份额提升+国产化视角；**MLCC线**由**王介超（金属新材料）**主导（上游镍粉/陶瓷粉材料视角），**杨晖（化工）**在6月从PTFE/氦气角度跟进。两条线的分析框架和团队完全不同。

## 时间线

### Phase 0：AI驱动PCB价值升级——从"通信周期股"到"算力核心材料"（2025年8月~11月）← 框架建立

**格局**：PCB在AI链中此前处于"有覆盖面但缺少独立叙事"的状态——常被归入光模块/服务器等大框架中作为配套环节。2025年8月起，随着景旺电子50亿元扩产、鹏鼎控股80亿元扩产，AI PCB作为独立投资主题开始成形。

**2025-08-28**：**AI PCB有望持续拉动PCB设备的更新和升级需求**（许光坦——机械团队首次系统覆盖）

**核心框架建立**：
- 全球PCB行业迎来新一轮上行周期：2024年全球PCB产值735.65亿美元，同比+5.8%
- AI服务器推动PCB高端化：18层板以上高多层板产值同比+40.3%，HDI板+18.8%
- 三大趋势：重回上行期、产品高端化、东南亚建厂
- **核心环节钻孔/曝光/电镀/检测价值量占比最高，AI对加工工艺提出更高要求**

**关键事件锚点**：
- 景旺电子50亿元珠海扩产（HDI/HLC/SLP）
- 鹏鼎控股80亿元淮安扩产

**2025-09-14**：AI PCB持续拉动设备更新 + 人形机器人密集催化（许光坦）
- AI PCB逻辑二次确认：北美PCB账面与账单比率连续8个月>1
- 设备端：钻孔/电镀/检测各环节工艺变化详解

**2025-11上旬**：**三团队年度策略联合锚定——PCB首次进入AI核心框架**

| 团队 | 分析师 | 年度策略视角 | 核心逻辑 |
|------|--------|------------|---------|
| **机械（设备）** | 许光坦 | AI PCB拉动设备更新和升级 | 产量增加+工艺变化→设备需求旺盛 |
| **电子（板厂）** | 刘双锋 | 云厂资本开支高增，PCB厂商积极扩产 | GPU+ASIC服务器PCB市场空间2025年超400亿、2026年超900亿 |
| **AI（国产化）** | 于芳博 | 国内PCB公司份额提升，带动上游国产化 | 高速CCL升级→树脂/玻纤布/铜箔国产化机遇 |

**2025-12-11**：**PCB产业链2026年投资前景**——年度策略正式发布
- 三团队联合深度：电子+AI+机械
- 覆铜板从M6/M7→M8/M9升级，带动上游树脂（PPO→碳氢）、玻纤布（E-glass→LowDk/石英布）、铜箔（RTF→HVLP4/5）同步升级
- **PCB钻针量价齐升逻辑**：窄线宽趋向+M9材料升级→单平米钻针用量+ASP双升

**叙事阶段**："AI驱动价值升级"——PCB作为AI算力"材料升级链"的定位自2025年8月起逐步建立，3个月内完成从零散跟踪到三团队联合覆盖的完整框架。

### Phase 1：PCB产业链深化 + 上游材料国产化展开（2026年3月~5月）← 子主题分蘖

**2026-03-11**：**PCB产业链投资前景**（三团队联合持续跟踪）
- 核心论点不变：AI驱动PCB价值量提升逻辑持续验证
- 深化上游材料分析：PPO树脂格局集中、LowDk玻纤布日本主导+中国追赶、HVLP铜箔壁垒高
- **Ruby正交背板**：英伟达Rubin Ultra将采用正交背板替代铜缆，3块26层PCB压合，M9+HVLP4+Q布高端材料组合

**2026-03~05月持续**：PCB扩产有序推进，但供需瓶颈短期难缓解
- 企业海外建厂超20亿美元，大陆建厂超300亿元
- 高端产能从投建到真实产能需经历工厂审核→产品认证→良率爬坡，无法井喷释放

**2026-06-08**：**《AI产业全景图谱》PCB价值重估——高速通信下的"隐形冠军"**（武超则等著）
- 中信智库+中信建投研究发展部联合出品（新书节选）
- PCB从"配角"升格为AI时代"隐形冠军"的定位

### Phase 2：MLCC超级周期——集中爆发（2026年5月24日~6月3日）← 最关键叙事切换

**2026-05-24**：**迎接MLCC超级周期，重视产业链投资机遇**（王介超——金属新材料团队）

**核心框架——MLCC的"超级周期"构建**：

**第一层：历史周期律**——MLCC呈现约15年的周期波动（村田经营数据）：
- 1980s：消费电子普及
- 2000s：通信设备升级
- 2010s：智能手机红利
- **2020s+：AI服务器+新能源车双轮驱动**

**第二层：需求三重爆发**：
| 需求端 | MLCC增量 | 核心逻辑 |
|-------|---------|---------|
| AI服务器 | +100%（传统服务器2倍） | GB200单系统MLCC 3000-4000颗，英伟达每代MLCC含量+50-60% |
| 新能源汽车 | 单车1.8万颗（燃油车6倍） | 纯电车MLCC用量是燃油车6倍，2030年车规MLCC超万亿颗 |
| AI PC/AI手机 | +40-60%/+20% | AI PC单机用量1400-1600颗，AI手机1300-1500颗 |

**第三层：供需矛盾确认**：
- 2026Q1 MLCC行业平均产能利用率87-88%，村田/三星电机>90%（>90%=价格上行通道）
- 高端MLCC供不应求，村田/三星电机/太阳诱电三家寡头垄断
- **涨价通道开启**：高端品种涨幅达15-35%
- AI服务器按颗数仅占全球1.1%，却占用7.5%产能——结构性紧缺

**第四层：上游材料传导**：
- 纳米镍粉：每亿颗MLCC消耗0.22吨，2030年需求从不足千吨→六千吨+
- 国内博迁新材-80nm镍粉已达全球顶尖水平，进入三星电机/国巨供应链

**2026-05-25**：**MLCC行业迎来"爆发时刻"**（周报头条推荐）
- 策略团队将MLCC列为"AI算力的细分高景气环节挖掘"三大主题之一
- 与"玻璃基板""炒股软件"并列

**2026-06-02/03**：**算力需求引发MLCC缺货潮**（杨晖——化工团队跟进）
- AI服务器MLCC搭载量较普通服务器增幅约3倍以上
- 需求激增导致供应紧张局面加剧
- 从化工材料角度：PTFE（用于高速PCB）有望大规模应用

**叙事升级**：从"PCB价值升级链"→ **"MLCC超级周期+PCB价值升级双轨并行"**

**叙事变化**：

| 之前（PCB主导） | 之后（双轨并行） |
|----------------|----------------|
| PCB是"价值升级" | MLCC是"超级周期"（更强周期性） |
| PCB扩产是年尺度的事 | MLCC缺货是季度尺度的事 |
| 只有许光坦/刘双锋/于芳博三团队 | 新增王介超（金属新材料）+杨晖（化工） |
| 上游材料是"国产替代"慢逻辑 | MLCC上游镍粉是"全球竞争力"快逻辑 |
| 叙事稳、框架细、持续时间长 | 叙事爆、数据密集、近3周内完成全部构建 |

## 当前叙事定位

MLCC/PCB处于 **"PCB价值升级+MLCC超级周期"双轨叠加期**——PCB的"AI材料升级链"（M6→M9/CCL→树脂玻纤铜箔钻针）逻辑仍在持续验证，MLCC的"超级周期"自2026年5月下旬刚刚爆发，缺货潮和涨价通道正在扩散。

**最新核心逻辑（2026-06）**：

**PCB线**：AI服务器架构从H100→GB200→Rubin持续升级 → PCB层数18-22层+ CCL材料M8/M9 → 覆铜板升级带动上游PPO树脂/LowDk玻纤布/HVLP铜箔国产化 → 钻针量价齐升 → 中国大陆PCB全球份额50%+持续提升

**MLCC线**：AI服务器MLCC用量3倍+传统服务器 + 电动车MLCC 6倍+燃油车 → 村田/三星电机产能利用率>90% → 高端MLCC结构性紧缺（1.1%颗数占7.5%产能） → 涨价通道开启（高端+15-35%） → 上游纳米镍粉/陶瓷粉需求倍增 → BB Ratio>1确认景气上行

**注意风险**：PCB新产能将从投建到良率爬坡存在不确定性；MLCC高端涨价可能抑制短期需求；MLCC超级周期是否重演2018年产能过剩历史。
