# 传导链工作台 (Transmission Chains Workbench)

> 从地缘事件→宏观变量→产业链传导的逻辑链框架
> 生成日期：2026-06-10
> 用途：捕捉"非标准推导"——那些不容易被常规研报覆盖但实盘价值极高的跨环节逻辑链

---

## 传导链模板

每条链包含：

| 字段 | 说明 |
|------|------|
| **起点** | 触发事件（地缘/政策/自然/价格信号） |
| **传导路径** | A→B→C→D→E 完整逻辑链，每层标注可观测指标 |
| **关键中间变量** | 链条上必须盯住的信号（如油价、运价、库存） |
| **终点** | 具体影响的产业链/品种/标的 |
| **相关叙事链** | 对应 narratives/timelines/ 中的产业链编号 |
| **验证条件** | 什么信号说明传导在生效 |
| **失效条件** | 什么信号说明链条断了 |
| **信源** | 验证过这条链的报告/数据 |

### Chain #001：霍尔木兹→原油→化工→日本断供→光刻胶/WF6

> 你的推导，被杨晖团队报告验证

- **起点**：霍尔木兹海峡关闭（2026年5月起已持续数周）
- **状态**：⏳ 传导中（中间变量已确认，终点品种价格待验证）

**传导路径**：
```
霍尔木兹海峡关闭（已持续数周，部分油井永久损伤）
  → 全球LNG贸易~20%受阻（卡塔尔RasLaffan受损，修复3-5年）
    → Brent油价$99.46，WTI$93.10，中枢抬升至$95-100+
      → 化工成本推升（打破"80美金以上化工无法传导"旧框架）
        → 日本全面依赖能源/化工品进口 → 生产成本飙升
          → 日本化工品供给收缩（光刻胶全球90%+份额在日本）
            → 特种气体（六氟化钨WF6）等细分品种短缺风险
              → 中国替代机遇
```

**关键中间变量**：
| 变量 | 当前值 | 数据源 |
|------|--------|--------|
| Brent原油 | $99.46/桶（环比+0.89%） | 杨晖6/9报告 |

| 美国原油库存 | 超预期去库798万桶 | 同上 |
| 美国钻井数 | 库存井+新钻井数创新低 | 同上 |
| 英国F-Gas法案 | 推迟削减→制冷剂高盈利久期延长 | 杨晖6/8报告 |

**终点**：
- 链#47 化工新材料（成本推升→盈利上修）
- 链#3 半导体材料（光刻胶/特气→短缺风险→国产替代加速）
- HALO型资产（通胀环境→高利润资产重估）

**验证条件**：
1. ⬜ Brent油价持续>$95/桶 —— 已部分验证（$99.46）
2. ⬜ 化工品价差指数从5.65%百分位回升（当前在底部，意味着一旦传导弹性极大）

4. ⬜ 日本化工企业减产/停产公告
5. ⬜ 杨晖团队判断"新一轮化工配置行情"兑现

**失效条件**：
1. ⬜ 霍尔木兹恢复通航（美伊达成协议/停火）
2. ⬜ Brent油价回落至$85以下
3. ⬜ 下游需求崩溃→化工品需求不足（衰退场景）
4. ⬜ 中国大量化工产能释放→抵消供给缺口

**信源**：

### Chain #003：AI颠覆焦虑→资金寻找不可颠覆资产→HALO价值发现

> HALO = Heavy Assets + Low Obsolescence，核心逻辑不是通胀，是AI对"护城河"的重新定义

- **起点**：AI对各行各业的颠覆加速（代码/客服/设计/金融等白领工作被替代）
- **状态**：⏳ 结构性的长期叙事，非短期交易

**传导路径**：
```
AI颠覆一切（LLM写代码/Agent做客服/AI生成设计稿/GPT处理法律文件）
  → 软件/互联网/信息资产的"护城河"被AI侵蚀
    → 数据壁垒不再有效（所有人都能用AI获取和处理数据）
      → 资金寻找技术迭代慢、AI颠覆不了的实物资产
        → 重资产（建厂300亿，AI不能帮你少花一分钱）
          → 低淘汰率（技术路线稳定，不存在被颠覆风险）
            → AI意外创造额外需求（算力+电力+冷却+MLCC+面板）
              → HALO资产获得"AI免疫+AI受益"双重溢价
```

**关键中间变量**：
| 变量 | 当前状态 |
|------|----------|
| AI替代白领的媒体报道密度 | 持续升温 |
| 软件/互联网公司估值 vs 制造业估值 | 剪刀差在收窄 |
| AI资本开支 (NVDA/MSFT/GOOG) | 还在扩张期 |
| 重资产行业资本开支意愿 | 仍然很低（供给约束持续） |
| 市场开始讨论"什么行业AI颠覆不了" | 正在成为投资话题 |

**终点**：
- 链#16 面板（京东方/TCL）
- 链#8 MLCC（风华高科/三环集团）
- 链#47 化工（巨化/万华/合盛）
- 链#43 战略金属（北方稀土/厦门钨业）

**验证条件**：
1. ⬜ 更多券商/策略团队提出类似HALO概念（叙事扩散）
2. ⬜ 重资产行业相对科技股的估值溢价走阔
3. ⬜ AI资本开支持续外溢到上游材料/元器件（需求驱动）
4. ⬜ 市场开始给"AI免疫"定价

**失效条件**：
1. ⬜ AI颠覆速度放缓→资金重新回流科技/成长
2. ⬜ 重资产行业出现技术颠覆（如固态电池颠覆电解铝需求——可能性极低）
3. ⬜ 全球深度衰退→所有资产普跌

### Chain #004：六氟化钨(WF6)涨价→半导体材料成本冲击

> 中信建投张玉龙2025年11月报告已有完整数据，霍尔木兹危机使其叠加日本供应风险

- **状态**：⏳ 第一波涨价已确认，第二波（日本成本冲击）待验证

**传导路径**：
```
第一层：钨原料价格快速上涨（独立于地缘，纯商品周期）
  → SK Specialty、Hoosung、关东电化等WF6供应商通知涨价70-90%

    → 关东电化（日本WF6生产商）生产成本进一步上升
      → WF6供给可能进一步收紧 → 价格二次上行风险
        → 半导体制造材料成本上升 → 产业链传导
```

| 变量 | 当前值 | 数据源 |

| 钨原料价格 | 快速上涨中（触发因素） | 同上 |
| 日本能源成本 | 受霍尔木兹→油价$99+影响飙升（2026年6月新增变量） | 杨晖6/9报告 |
| Brent油价 | $99.46/桶，中枢上行 | 杨晖6/9报告 |

- 链#3 半导体材料（WF6是关键的CVD沉积特种气体）

**验证条件**：

**失效条件**：
1. ⬜ 霍尔木兹恢复通航→油价大幅回落→日本能源成本正常化
2. ⬜ 半导体需求衰退→WF6需求下降
3. ⬜ 中国国产WF6产能快速释放→替代进口→供给充裕

**信源**：
