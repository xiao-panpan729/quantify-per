# 叙事判定层 — S/A/B/C 分级

> 判定标准：**叙事转折（narrative inflection）** 优先——不是增速高低、故事好坏，而是叙事框架本身近期是否发生本质改变，以及是否有资金流入验证。
> 
> 生成日期：2026-06-10
> 基于53条叙事时间线 + 500字模板通读

---

## 分级定义

| 级 | 含义 | 特征 |
|----|------|------|
| **S** | 叙事正在转折 | 框架本质变化（3个月内）+ 资金流入确认 + 分析师集中覆盖 |
| **A** | 叙事清晰上行 | 框架已成立 + 持续催化 + 资金稳步流入 |
| **B** | 叙事稳定/磨底 | 框架未变 + 周期位置明确（底部/稳态/过渡） |
| **C** | 叙事弱化/空窗 | 无新叙事 + 催化剂缺失 + 资金流出/观望 |
| **D** | 叙事消亡 | 旧框架已破 + 新框架未立 + 行业收缩 |

### 实操指引（操作态度差异化）

| 级 | 操作态度 | 入场耐心要求 |
|----|---------|-------------|
| **S/A** | **多花时间看，操作可更积极** | ABC回调的A调整有信号可考虑，不必等完整闭环 |
| **B** | **正常跟踪，等催化** | 多等信号——等缠论ABC调整背驰+★闭环，或多次★买闭环才入场 |
| **C** | **没有强逻辑不碰/少碰** | 必须多等——ABC调整后等至少2个5分钟★闭环信号确认 |

**要点**：
- S/A 级不是无脑追，而是"信号门槛降低"——因为叙事转折提供了额外安全边际，ABC回调的A段出信号就可以考虑，不必死等C段完整
- B 级回到标准流程——缠论 ABC 调整背驰 + ★闭环 / 多次★买闭环确认
- C 级门槛最高——即便有信号也等多一次闭环确认，宁可错过不可做错
