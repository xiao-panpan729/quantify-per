# quantify-per — 量化交易信号系统

> 基于通达信数据的多周期量化信号计算与 AI 报告生成系统。
> 自动跟踪 11 只标的，生成信号快照和智能分析报告。

---

## 🚀 快速开始

```bash
# 盘后运行（通达信下载完数据后）
run_daily.bat
```

自动完成：数据同步 → 信号计算 → 快照保存 → AI 报告

## 📊 当前功能

| 模块 | 状态 | 说明 |
|------|:----:|------|
| 数据同步 | ✅ | 通达信 `.day/.lc5` → pytdx → CSV 缓存 |
| 多周期信号 | ✅ | 分时出击 + EXPMA + CCI + 机构建仓 |
| 趋势强度分级 | ✅ | A/B/C/D 四级判断 |
| 筹码选股 | ✅ | 93.1% vs 通达信（`chips_selector.py`） |
| AI 智能报告 | ✅ | 状态叙事结构，`trading_persona.md` 控制风格 |
| 预检机制 | ✅ | 同步前自动验证数据完整性 |
| 时间戳缓存 | ✅ | float 精度，避免缓存误判 |
| 闭环检测引擎 | ✅ | CCI极值+背驰+★买/★卖+EXPMA交叉闭环评分 |
| 多级别嵌套分析 | ✅ | 日线→60分→30分→15分→5分逐层定性 |
| 反向信号识别 | ✅ | 连死叉→横盘→金叉转折点判定 |
| 多级共振检测 | ✅ | 同方向多级别同时闭环=共振 |
| 闭环数据归档 | ✅ | JSON格式供回测使用 |
| 回测框架 | 🏗️ 设计中 | Phase 1 日志 → Phase 4 策略优化 |
