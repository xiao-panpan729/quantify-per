            ├── 旧: 偏离即卖（与趋势脱钩）

2026-03-20  [新增] 短线退出机制 4 级
            └── 多周期共振死叉 + 突发事件密集

            └── 下跌反弹中卖信号≥2 + 偏离≥2% → 即使 EXPMA 不死叉也应做 T

            ├── DIF 和 DEA 都在 0 轴上方 = 多头，否则 = 空头
            └── 顺势简单确认，逆势多次确认
            
2026-04-07  [重大发现] 信号层/显示层分离
            ├── 旧误区: 趋势线交叉和 C>midline 混为一谈
            └── 新框架: 信号层（交叉）≠ 显示层（位置）
            
2026-04-08  [新增] 周线参数规则
            └── LLV/HHV 参数 N=40（不是日线的 55）
            
2026-04-08  [新增] 色块定义
            ├── trend<11 = 黄色区（底部蓄力）
            └── trend>89 = 白色区（顶部蓄力）
            
2026-04-08  [新增] 周线双信号加强
            └── 两次周线★买间隔出现 = 反向加强
            
2026-04-15  [验证] 趋势线算法 N=55 T=91 最佳
            └── 520600 日线前 4 个趋势线值与通达信完全一致
            
2026-04-20  [根因确认] lc60 N=40（非 N=55）
            ├── 旧: 所有周期统一 N=55 → lc60 偏差巨大
            └── 新: lc60 N=40 → diff=0.000 完美匹配
            └── 教训: TDX 公式名"分时出击(40)"中的 (40) 就是 N
            
2026-04-21  [统一] 日线/min15/min30 N=55 统一；lc60 N=40 独立
            
2026-04-22  [重大纠正] CCI 方向纠正
            ├── 旧错误: ★买→CCI正极值，★卖→CCI负极值（说反了）
            └── 新正确: ★买→CCI负极值(≤-200)向上回撤；★卖→CCI正极值(≥+200)向下回落
            └── 完整闭环 = CCI极值 + 背驰 + ★买 + EXPMA金叉（三步缺一不可）
            
2026-04-22  [新增] 两个闭环分工
            ├── 大周期(30min): 找第一次闭环，日线不强则只观察不动
            └── 小周期(5min): 找第二次闭环，极值+★买+金叉 = 试错点
            
2026-04-27  [认知升级] 趋势强弱优先于信号
            ├── 旧: 看到"底背驰+★买"就建议试错
            └── 新: MACD 0轴下方 = 弱势，第一次金叉不可信，等二次探底+二次金叉
            
2026-04-28  [新增] 信号级别匹配原则
            ├── A最强(白线上方+MACD>0) → 5min可操作
            ├── B次强(白线-黄线间+MACD>0) → 5min可操作
            ├── C偏弱(黄线下方但MACD>0) → 15min（5min反弹不可信）
            └── D弱势(MACD<0) → 30min+（小级别反弹只是反弹）
            
2026-04-28  [新增] 基本面判断原则
            ├── 趋势多头 → 关注回调后买入机会（不应"强烈提示风险"）
            └── 风险条件: 60min★卖+CCI背驰 或 到达前期高点压力位
```

### 2.3 TRD — 交易哲学

**核心理念**: 大预期小操作、该跌不跌、大盘优先、中国资产。

```
2026-03-12  [新增] 做 T 逻辑 + 大盘优先 + 该跌不跌
            ├── 所有做 T 先看大盘环境
            ├── 大盘没走好 → 个股容易多挨震荡
            └── 该跌不跌 = 跌不动就要涨（多头力量强于预期）
            
2026-03-13  [新增] 过头翻翻法则
            
2026-03-16  [新增] 大预期小操作
            ├── 大预期: 周线类二买、结构支撑
            ├── 小操作: 30 分钟反弹、10-15 点目标
            └── 怀抱大方向信心，执行小级别操作
            
2026-03-16  [新增] 中国资产（宏观视野）
            └── "世界回不到过去了" → 中国资产成为必要分散配置
            
2026-04-09  [新增] 信号三阶段决策框架
            ├── 等待动手: 0 仓位，只盯盘
            ├── 试错买入: 1-3 成仓位
            └── 正式建仓: 7-10 成仓位
            
2026-04-22  [新增] 操作纪律（子层级六，基于 159740 实盘教训）
            ├── 回补仓位管理: 回补≠必须平掉，走对继续拿，走错止损
            ├── 跳空日纪律: 跳空低开当天禁止买入
            ├── 多次死叉预警: 5min两次死叉→预警；跳空+第三次死叉→调整确认
            └── 日线空头环境补仓限制: 空头下补仓有效期更短、容错率更低
            
2026-04-23  [新增] 三张赌桌理论
            ├── 先考虑风险（第一张赌桌防御）
            ├── 小赚积累（一致性收益）
            └── 达到一定阶段后进攻（第三张赌桌捕捉极端机会）
            
2026-04-26  [新增] 补仓哲学（从 USER-CORE 同步）
            ├── 补仓是中性的
            ├── 走对了继续拿
            └── 走错了止损（跳空+新信号如第三次死叉）
