## 信号密度改为波段计数（indicators.py signal_quality）
- **原因**：全窗口平均（"3次/448根K线"）没有意义，密度应该按"这一段下跌/上涨"算
- **买侧**：`buy_wave_start` = 最近一个★卖或死叉的位置，只计此之后的★买
- **卖侧**：`sell_wave_start` = 最近一个★买或金叉的位置，只计此之后的★卖
- **阈值**：≥3次/波=密集+1.5 / 2次/波=连续+1.0 / 1次/波=单次+0.5 / 0=0
- **标签**：从"X次/NNN根K线"改为"X次/波"

## 量能确认重构：单维度→三维度（indicators.py signal_quality）
- **原因**：旧逻辑"有地量堆就+1.5"太粗糙，无法区分缩量程度
- **前提**：趋势线<10（超卖区）才激活
- **三维度各0.5，共1.5**：
  1. 百日地量出现在★买附近 → +0.5
  2. 地量堆存在于波内 → +0.5
  3. 地量堆均量 ≤ 百日地量均量×120% → +0.5
- **非超卖区**：维持旧逻辑（放量/缩量/梯度放量）
