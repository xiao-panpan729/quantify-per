# 2026-05-22 变更日志

## 新模块：成交额强者选股器 (tools/volume_leader_screener.py)

灵感来自胜宏科技（sz300476）走势：持续创新高 + 成交额持续排名前50。

### 双维度筛选逻辑

**条件A — 成交额排名**
- 扫描全A股~4600只，读每只 `.day` 文件最后32字节获取当日成交额
- 全市场按成交额降序排列，取 Top50
- 持续排名：每日盘后 `--update-rank` 追加快照到 `volume_rank_history.csv`，后续可查"近20日中有多少天在Top50"

**条件B — 后复权创历史新高**
- 数据源：本地 `.day` 日线 + `gbbq/gbbq.csv` 除权数据
- 后复权：正确处理分红/送股/配股，历史最高价经复权调整后与当前价可比
- "创新高"定义：后复权收盘价达到或超过全上市历史最高价（容差0.1%）
- 窗口检查：近N个交易日（默认20）内是否出现过新高

### 关键参数

| 参数 | 默认 | 含义 |
|------|------|------|
| `--top 50` | 50 | 成交额排名前N |
| `--window 20` | 20 | 新高检查窗口（交易日） |
| `--update-rank` | - | 先更新排名快照再筛选 |
| `--fetch-names` | - | 一次性从pytdx拉全量名称缓存 |

### 输出格式

```
★★★ 极品标的 (Top50成交额 + 近20日创新高): N 只
排名  代码        名称        现价    成交额(亿)  持续Top50日  距新高%  新高次数  最近新高日
1     sh603986   兆易创新     431.90   282.0      1/1日       -3.0%   1/20日   2026-05-20
...
关注标的 (Top50成交额，但20日内未创新高): M 只
...
```

### 性能

- 全A股成交额扫描：0.5秒（4603只，只读最后32字节）
- 后复权新高检查：每候选股 ~0.1秒（只对Top50计算，非全市场）
- 全流程：6-8秒
