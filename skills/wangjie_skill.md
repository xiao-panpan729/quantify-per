# 王姐Skill — 短线实战专家

**来源**: 申港证券王会丹（执业编号S1660625080001）2025年4月-6月复盘笔记33篇精读合成  
**定位**: 短线情绪交易 + 机构趋势两线并行框架  
**覆盖周期**: 完整市场周期（台子搭建→主升→分歧→退潮→底部酝酿）  
**精读日期**: 2026-06-14

---

## 一、核心哲学

### 1.1 三条铁律

1. **环境第一位，个股第二位** — 大盘环境向下，再好的股也要先走。环境向上，个股弱一点可以等。
2. **身份重于价格** — 相同的价位，选身位高的。板块中军的稳定性远高于跟风股。
3. **一个方向深耕，不要来回切** — 试错段选一个方向埋种子，确认后加仓。来回切换亏的不比退潮少。

### 1.2 两线分离

```
机构线                         情绪线
──────────                    ──────────
看业绩/确定性                  看故事/想象力
日K线操作                      分时/连板操作
慢节奏（3-5天）                快节奏（1-2天）
同体量切换（大→大）            题材试错接力
大中军不死=行情在              连板高度=情绪温度
```

**永远不要用情绪流的思路理解机构，不是一套体系不要硬融合。**

### 1.3 三段论（元框架）

每轮大行情的标准剧本：
```
第一段：机构搭台 → 确定性业绩方向（如CPO/PCB）领涨
第二段：情绪唱戏 → 题材股接棒，赚钱效应扩散
第三段：赶顶 → 全面亢奋后见历史大顶
```
不是每轮都完整走三段，但框架提供基准。偏差时知道偏差在哪里。

## 二、市场阶段判断体系

### 2.1 五阶段划分

| 阶段 | 特征 | 对应时间（案例） |
|------|------|-----------------|
| **① 试错段/台子搭建** | 机构先动、情绪跟不上、量能温和放大 | 4.8-4.22 |
| **② 主升段** | 量价齐升、百股涨停、连板梯队完整 | 4.29-5.14 |
| **③ 分歧/切换段** | 高位放量滞涨、科技分化、新题材试错 | 5.15-5.28 |
| **④ 退潮段** | 核心票放量下跌、量缩、无主线 | 6.1-6.11 |
| **⑤ 底部酝酿段** | 极致缩量→核心票补跌→大阳确认 | 待验证 |

### 2.2 核心判断指标（按优先级）

```
1️⃣ 量能 → 缩量/放量 + 方向（最重要的信号）
   - 放量下跌 = 新做空力量
   - 极致缩量（<2.5万亿） = 空头衰竭
   - 量缩不动 = 止跌信号

2️⃣ 情绪vs指数匹配度
   - 持续背离 = 迟早要修复或大回落
   - 共振向上 = 最健康状态
   - 情绪先于指数止跌 = 市场即将修复

3️⃣ 机构压舱石状态
   - 中际旭创不死 = 科技行情在
   - 核心票补跌 = 退潮进入尾部

4️⃣ 连板生态
   - 高度+数量+梯队完整度 = 情绪温度计
   - 高标断板→看次高卡位→看板块接力
```

### 2.3 退潮确认链

```
前奏：科技大跌→次日强修复（1-2次正常）
       ↓
警示：再次大跌→修复减弱（注意了）
       ↓
确认：核心大票（中际旭创/寒武纪）放量下跌
       ↓
加速：全线跟跌，量能不缩
       ↓
尾部：最坚挺的票都补跌了，量缩到极致
       ↓
底部：要么阴跌不动，要么恐慌盘出清后量缩
```

## 三、身份等级体系

### 3.1 身份金字塔

```
身份等级        含义                         操作模式
──────────      ──────────                   ──────────
全市场核心      不能死，决定行情存亡           中际旭创、寒武纪级别，它们崩了行情结束
  ↓
板块大中军      板块定海神针，风向标           卫星、英维克、海光、华虹
  ↓
趋势引领       有身位，带领板块走              西部材料、神剑、宏和科技
  ↓
一波流高度     连板打空间，情绪锚              博云新材、圣阳、香江控股
  ↓
300套利        弹性工具，不格局                信维通信、飞沃、甬矽电子
  ↓
纯套利         量化筹码/边角料                 金螳螂、宝丽迪
```

### 3.2 身份升级机制

- **扛过来 = 身份升级** — 板块分歧时能扛住不跌的个股，身份升级为下一段的核心
- **断板/破位 = 身份降级** — 打出强辨识度后跌停，"身份就被搞掉了"
- **大票反弹有力 = 真想去二波** — 不是混1-2天就跑

### 3.3 题材审美三要素

一个题材要成为主线必须满足三个条件：
1. **有故事讲** — 深入人心，认同越多发育越大
2. **机构能参与** — 有业绩或产业逻辑支撑
3. **容量够大** — 能承接大资金的体量（3万亿市场，十几只小票的题材直接PASS）

## 四、各阶段操作手册

### 4.1 试错段/台子搭建

**要做的事**：
- 跟机构方向配置（CPO/PCB/设备），"不喜欢多动就躺"
- 同时埋种子：选一个题材方向小仓位试错
- 操作节奏：加速2-3天先出，回调1-2天尾盘低吸

**不要做的事**：
- 频繁切换方向（"来回换亏的不比退潮少"）
- 用情绪流思维做机构票
- 300套利上头

**核心规则**：
- 低吸看日K，不看分时
- 10日线上方运行才安全
- "边打边撤是主基调"+ "他不会挂掉" = 同一板块内的高切低

### 4.2 主升段

**确认条件**：量能持续放大 + 上涨家数多 + 连板梯队完整 + 情绪未亢奋

**要做的事**：
- 大仓位跟随引领品种
- 情绪流选手选趋势大容量票（体量大+趋势好+容易活）
- 主线（科技）不动，支线（航天/机器人）并行布局

**不要做的事**：
- 亢奋后追高（"市场加速就是赶顶"）
- 把反弹当主升
- 忽视上面压盘信号

**核心规则**：
- "第一次大分歧不用担心→捡筹码机会"
- "没爆量杀恐慌盘/没极致高潮→不看空做空"
- 主升时不乱切换，"坚定看好一个方向"

### 4.3 分歧/切换段

**要做的事**：
- 关注"扛过来"的个股/板块 → 身份升级 → 成为下一段载体
- 高切低：从高位领涨切换到低位新方向
- 见好就收：CPO/PCB逐步减仓

**核心判断**：
- **同级别替代原则** — 机构资金只能由同体量承接（CPO水位→半导体先进制程，小票接不住）
- **越往后走，新承载体更具弹性**

**关键信号**：
- 核心票高位滞涨≠死，可能要换领涨
- 科技第一次松动→后续缩容炒作
- 大票反弹有力=真想去二波

### 4.4 退潮段

**确认条件**：核心大票（中际旭创/寒武纪）放量下跌

**要做的事**：
- **急跌期（1-3天）**：不动，等主跌期过
- **反弹期**：科技反弹=出局机会；超跌反弹可轻仓博弈
- **尾部（核心补跌）**：可以准备动手了

**能做的方向**：
- 穿越活口（小仓位套利）
- 超跌反弹（科技对立面：消费/电力/商业航天）
- 涨价线（有业绩支撑）

**千万不能做的事**：
- ❌ 满仓冲
- ❌ 科技大票（反弹是出局，不是入场）
- ❌ 把反弹当主升
- ❌ 追高前期热点

**底部信号**：
- 要么缩到极致跌不动
- 要么砸出巨量恐慌盘然后逐步缩量
- 核心票补跌 = 赶底特征

### 4.5 震荡/底部酝酿段

**特征**：量缩到极致（<2.5万亿），核心票补跌，无主线  
**操作**：等大阳线确认 → 大阳出现左侧布局  
**注意**：退潮段出现的所有板块都只是超跌反弹，"不可能有主线"

## 五、规则引擎

### 5.1 买入规则

| 条件 | 操作 |
|------|------|
| 市场上升 + 分歧缩量 | 低吸机会 |
| 科技分化时 | 回调1-2天后低吸封测/半导体 |
| 东阳光开板 | 低吸模式 |
| 天通/云南锗业深水 | 缠打模式（深水没问题） |
| 情绪龙头 | 选趋势大容量票 |
| 超跌反弹阶段 | 上半周超跌方向，下半周题材 |
| 止跌信号出现 | 等放量上攻确认 |

### 5.2 卖出/止损规则

| 条件 | 操作 |
|------|------|
| 环境向下改变 | 先走，环境第一位 |
| 个股主升段2根阴线 | 弱了，考虑走 |
| 破10日线 | 转弱确认 |
| 高位放量大阴线 | 建议走 |
| 身份已没落（前身位） | "该出就出" |
| 星链事件临近（6.12） | 越靠近预期越小→逐步减仓 |
| 科技每次修复 | 撤的时候 |
| 不愿意忍受波动 | 出，到低吸点会通知 |
| 走加速且开5%以上 | 上看一眼都是错 |

### 5.3 不看/不参与清单

- 百亿市值以下的票（"基本不看"）
- IDC全线弱化→"这条线不建议玩了"
- 电力逆指数→不主推，不上仓位
- 金螳螂/宝丽迪→涨10倍也不是龙头
- 算力租赁→天然缺陷（机构不认）
- 机器人中军不涨→机构不感冒→难成主线
- 情绪没主升时不存在确定主线

### 5.4 仓位管理

| 市场阶段 | 仓位建议 |
|---------|---------|
| 试错段 | 轻仓埋种子，等市场选方向 |
| 主升段 | 重仓引领品种 |
| 分歧段 | 减高位，试低位 |
| 退潮段 | 轻仓/空仓，小仓位穿越+超跌反弹 |
| 底部酝酿 | 可以开始布局 |

## 六、强势股识别 — "身位就是战斗力"

### 6.1 怎么看身位

1. **先看大中军** — 破位=板块出问题。大中军不死=行情在
2. **再看引领者** — 谁在带头涨？有没有持续引领？
3. **看板块梯队** — 有没有完整的连板梯队？高标断板有没有次高卡位？
4. **看大票表态** — 大市值票涨停=机构认可，板块真想博弈二波

### 6.2 真突破 vs 假突破

| 真突破 | 假突破 |
|--------|--------|
| 带量确认，后续有承接 | 无量空涨，放量冲高回落 |
| 大市值票涨停（机构认可） | 全是小票在嗨 |
| 板块效应明显 | 个股行为，板块不动 |
| 引领品种持续性强 | 前排天天换，没有持续性 |

### 6.3 核心判断口诀

- "中军涨势一般=机构不感冒" → 板块难成主线
- "前排走不动/隔三差五换新→板块在套利" → 不参与
- "情绪流只看身份，不看行业身份" → 谁最强看谁
- "身份可以被摧毁" → 打出辨识度后自己跌停=身份没了

## 七、关键信号监控表

| 信号 | 含义 | 后续动作 |
|------|------|---------|
| 中际旭创放量下跌 | 科技退潮确认 | 减仓/出清科技线 |
| 证券拉升+科技滞涨 | 证券-科技背离 | 科技可能要砸 |
| 极致缩量（<2.5万亿） | 空头衰竭 | 准备动手 |
| 情绪先于指数修复 | 市场即将止跌 | 观察新方向 |
| 百股涨停+情绪未亢奋 | 主升健康 | 继续持股 |
| 连板高度下降+数量<10 | 情绪小冰 | 等回暖 |
| 核心票补跌 | 退潮尾部 | 准备布局 |
| 大阳线+放量 | 反转确认 | 加仓 |

## 八、王姐Skill调用方式

> **定位**: 输入当前市场的量能、情绪、板块结构数据 → 输出市场阶段判断+操作建议  
> **适用**: 每日复盘/盘前决策  
> **依赖**: 市场数据（量能/涨跌家数/连板数/板块表现）+ 核心个股状态

### 使用示例

输入: `当前量能? 连板高度? 核心票中际旭创/寒武纪状态? 机构线状态? 情绪线状态?`  
输出: `当前市场处于[阶段]，机构线[状态]，情绪线[状态]，建议[操作]...`
