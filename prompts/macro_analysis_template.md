# 宏观分析输出模板 — Dorian 框架 + 洪灏变量拆解

**来源**: 灰岩金融科技 Dorian 君（六步流程）+ 洪灏（变量拆解/逆向思维）
**定位**: 六层宏观数据 → 六步结构化分析 → 人味决策总结
**适用场景**: 
  - 用户问"宏观分析"/"市场怎么样" → 跑六层数据，按此模板输出
  - `python gen_source_summary.py --ai` → AI日报也走完整六步，不是只输出观点

## 数据完整性自检（动笔前必做）

在走六步之前，先检查哪些数据源是空的/旧的：

1. 逐一确认上面六层数据每个都有值
2. 如果某层数据缺失 → 推理"正常情况下这里应该有什么"，在分析中标注
3. 如果shock_detector为空但市场明显在定价某个主题 → 写"XX事件可能未被数据源捕获"
4. 如果公众号为空 → 写"缺少7个信源当日观点交叉验证"
5. 如果多处缺失 → 降低结论置信度，在开头标注"本文基于不完整数据"

### 第0.5步：叙事演化定位（横向+纵向，分析前必做）

**在走六步流程之前，先定位今天的叙事在时间轴和历史截面上的位置。**

**横向（历史进程）** — 核心叙事在时间轴上怎么演的：
1. 读最近2-3天的信源日报，回答：
   - 昨天/前天的核心叙事是什么？
   - 今天这个叙事是强化了、弱化了、还是转向了？
   - 有没有新叙事在浮现？旧叙事在消退？
2. 标注当前状态：`一致性/分歧/转向/混沌`

**纵向（事件叠加与交叉）** — 当前时间节点上多事件的交汇效果：
1. 把今天同时发生的事件全列出来
2. 它们是互相强化、互相抵消、还是各自独立？
3. 叠加后的合成方向是什么？

**输出方式**：这个定位结果写在因果链展开之前，1-2句定性即可。

## 输出格式（两部分）

### 第一部分：六步结构化分析（Dorian完整链）

```
## 宏观框架分析 — YYYY-MM-DD

### 1. 变量拆解（洪灏：找到"最关键的变量"）
列出当前所有关键变量及其方向（↑↓→），用图标标记极端值（🔥）

洪灏法则：不要罗列所有变量，找到当前周期里"最关键的1-2个变量"。
  问自己：如果只能选一个变量解释当前市场，是什么？
  因为市场在任何一个时间点，定价的只是1-2个核心矛盾，不是所有变量。

### 2. 传导链（洪灏链式推导）
用箭头画出变量之间的因果传导，必须能追溯到某个可观测数据的变动。

传导链必须完整走完：
  A发生了 → 为什么A导致B → B传导到C → C意味着什么

每层转折问"所以呢"：
  "油价涨了 → 所以通胀预期上升 → 所以利率路径收紧 → 所以成长股估值压缩"
  不是"油价涨了，通胀压力"（这是信息罗列，不是链式推导）

### 3. 历史对标
找历史上类似变量组合出现的时期，说清楚"像在哪里、不像在哪里"

三种历史对标模式：
  - 宏观组合对标（类似通胀+利率+增长组合）
  - 事件对标（类似地缘/政策冲击）
  - 市场结构对标（类似流动性环境+波动率结构）

对标时强制回答：这次"不一样"的地方在哪里？
  因为市场永远在定价"与历史的不同"，而不是"与历史的相同"

### 4. 情景推演（分叉树）
三个情景：基准(概率最高)、乐观(什么条件触发)、悲观(什么条件触发)
每个情景一句话说明对应的市场表现

关键：情景之间必须互斥、穷举
  ❌ "可能涨也可能跌" → 这不是情景，这是废话
  ✅ "如果Fed因通胀暂停降息 → 利率上行 → 成长股承压；如果Fed因增长放缓提前降息 → 美元走弱 → EM受益"

### 5. 交易映射
不是给具体买卖点，而是说清楚：
  - 现在该进攻还是防守（仓位倾向）
  - 哪个方向相对占优（价值/成长、大盘/小盘、A股/港股）
  - 什么信号出现后可以改变判断

### 6. 失效条件 ← 最重要的一步（Dorian）
强制写：什么情况下"我上面的分析是错的"

失效条件的四种类型（至少写2种）：
  ① 数据失效：某个关键变量突破了什么阈值
     "如果CPI连续两月高于3.5%，我的'通胀降温'判断失效"
  ② 逻辑失效：传导链中的一环被证伪
     "如果油价上涨但美债收益率不跟，说明我在定价的不是通胀逻辑"
  ③ 时间失效：过了窗口没发生
     "如果两周内Fed没有释放鸽派信号，我的'即将降息'判断不成立"
  ④ 结构失效：市场结构变了
     "如果波动率持续下降但风险资产也在跌，说明不是risk-on/off，是流动性问题"

判断标准（自检）：
  - [ ] 失效条件是具体的"如果X则Y"，不是泛泛的"如果市场变了就错了"
  - [ ] 失效条件里有可观测的阈值或变量（CPI/利率/汇率点位）
  - [ ] 没有失效条件的分析 = 不是交易判断，只是观点
```

### 第二部分：人味总结（叙事式）

```
## 市场叙事

用3-5段写一个连贯的市场故事，要求：
- 不列bullet point，用自然段落
- 有观点、有判断，不是"一方面...另一方面..."
- 敢说"现在最大的风险是X"或"市场在错误定价Y"
- 用比喻让复杂关系变得直观（但别过度修辞）
- 长度：手机一屏能读完（300-500字）
- 结尾给一个清晰的倾向性判断
- 开头写今天最重要的1个锚点事件（不是综述）

风格参考：
  "BTC暴跌35%不是孤立事件——它是全球流动性的金丝雀。
   当金丝雀不叫了，矿工就该检查氧气面罩了。"
  ← 这种风格可以，数据+画面感，但不要每句都这样

洪灏风格参考：
  找市场最大共识 → 问"这个共识哪里可能错了" → 给出反直觉判断
  例子："所有人都在说通胀会回落，但如果通胀的粘性不在商品而在工资呢？"
```

## 执行规则

1. **必须先跑数据** — 用最新 JSON 文件，不凭空分析
2. **变量拆解要找"最粗的那条线"** — 不是罗列，是找到当前市场定价的核心矛盾
3. **传导要有方向** — 箭头链必须能追溯到某个可观测变量的变动，不能停在"不确定性"
4. **失效条件必须写** — Dorian原话："交易最重要的一步是，什么时候我错了。"没有失效条件的分析不输出
5. **人味总结要敢下判断** — 不说车轱辘话，要有观点。判断错了下次复盘就好，没判断等于没分析
6. **不要当数据搬运工** — 引用数据自然融入叙事，不是"CPI是X%，M2是Y%"的生硬堆砌
7. **关注超级IPO与流动性抽血** — 未来1-3个月有没有超大市值IPO/增发/解禁？SpaceX 6/12上市募资750亿刀、Anthropic/OpenAI排队上市，这些直接抽取市场流动性，影响往往超过宏观因子。问自己：IPO抽血进不进入因果链？指数再平衡压力多大？
8. **关注跨境资本流动的监管变化** — 老虎/富途境外账户清退（6/12起允许卖不允许买）、QDII额度持续收窄、人民币汇率变化。时间点有没有巧合？清退令6/12生效 vs SpaceX 6/12上市，这不是巧合，是精准卡位。
9. **读历史信源日报保持一致性** — 写之前去 reports/sources/ 翻过去1-2周的信源日报，看之前判断的核心矛盾是什么，有没有变化，失效条件有没有触发。
10. **最终闭环：交易员检验** — 写完分析后问自己："这份数据如果给一个交易员看，他接下来会做什么动作？" 结论不能是"等待时机/谨慎乐观"这类正确废话。必须有一个可观察、可证伪的方向判断。

## 分析态度检查（输出前自测）

每条分析输出前问自己：
- 我有没有找到"最关键的1-2个变量"？（洪灏法则）
- 我的传导链走完了"所以呢"的拷问吗？（链式推导）
- 失效条件定了几个？是可观测的具体条件吗？（Dorian）
- 我的结论是"一边…一边…"的和稀泥，还是明确的判断？（人味）
- 如果把这篇给一个专业交易员看，他会觉得有价值还是浪费时间？

## 验证清单

输出前自检：
- [ ] 六步都填了，不是空的
- [ ] 横向（历史进程）定位做了吗？叙事是强化/弱化/转向？
- [ ] 纵向（事件叠加）分析做了吗？多事件是共振还是对冲？
- [ ] 变量拆解找到了核心矛盾，不是罗列
- [ ] 传导链至少跨了3层（宏观→流动性→市场→板块）
- [ ] 给了至少2种具体的失效条件
- [ ] 给了一个明确的方向倾向，不是"高抛低吸"
- [ ] 失效条件不是泛泛的"如果市场变了就错了"
- [ ] 人味总结里没有"一方面...另一方面..."
- [ ] 人味总结开头是一个具体事件，不是综述
- [ ] 数据完整性已检查，缺失的数据源有做缺失推理
- [ ] 超级IPO抽血已审视（近期有没有超大IPO/增发？）
- [ ] 跨境资本管制已审视（资金出海通道有无变化？时点巧合分析？）
- [ ] 如果数据源缺失，结论置信度已做调整
- [ ] 历史信源日报已查阅，观点一致性已检查
