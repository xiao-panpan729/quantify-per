# 信源分析写作模板
# 每次 update_sources.bat 运行时，AI 按此模板输出当日市场分析（输出文件：信源日报 reports/sources/YYYYMMDD_sources.md）

## 前置流程（分析必须走完六步，再按五段输出）

### 第-1步：读过去信源日报（保持观点一致性）

**动笔前，先读过去1-2周的信源日报（reports/sources/ 目录），回答以下问题：**

1. 前几天我判断的"核心矛盾"是什么？今天有没有变化？
2. 前几天我写的"失效条件"有没有被触发？如果触发了 → 承认判断错了，重新分析
3. 有没有之前提到的叙事今天在延续？→ 保持一致性，不要每天换一个说法
4. 有没有之前我完全没提到的新变量出现？→ 标注"这个变量是本周新出现的"

**特别提醒**：如果之前的判断被市场验证了，直接说"X日判断的Y正在兑现"。如果判断错了，明确说"此前判断的Y没有发生，原因是Z，修正为W"。**不要假装之前的判断不存在，也不要避讳认错。**

### 第0步：数据完整性自检（动笔前必做）

在走六步之前，先检查数据源完整性。逐一确认：

```
本次实际收到的数据源（从摘要里看哪个区块有东西/哪个是空的）：
1. 微信公众号文章 → 有/空 → 如果空：MPTEXT API可能过期了
2. 消息面冲击(shock_detector) → 有/空 → 如果空：WS+东财API可能超时
3. 全球流动性(liquidity_monitor) → 有/空
4. 中国宏观快照(macro_snapshot) → 有/空
5. US宏观(us_macro_sensitivity) → 有/空
6. 日本宏观+套息(japan_macro) → 有/空
7. US ETF动量(us_sector_momentum) → 有/空
8. US明星股动量(us_star_momentum) → 有/空
9. 概念链轮动(us_concept_momentum) → 有/空
10. 基本面因子(fundamental_profile) → 有/空
```

**数据缺失推理规则**（这是最重要的一步——空数据不等于没有事情发生）：

- 如果公众号为空 → 缺失了7个信源的当日观点。你无法知道猫笔刀/表舅/卓哥/亨特/laoduo今天在讨论什么。**推理方向**：结合你已有的宏观数据，推测公众号可能关注什么话题
- 如果shock_detector为空 → 缺失了当日突发事件检测。**推理方向**：检查其他数据源中有无异常值（VIX飙升/汇率跳变/板块暴跌），反向推断可能出了什么事
- 如果某ETF/明星股数据为空 → 缺失了日度动量排名。**推理方向**：使用趋势延续假设计算"如果有数据会是什么"，标注为推测
- 如果多个数据源同时为空 → 请标注"本文基于不完整数据"，降低结论置信度

推理结果写入文章第1段（锚点事件之后）或第4段（多源交叉验证），用一句说明："本文数据源X缺失，以下YY部分是推测，请交叉验证"

### 第0.5步：叙事演化定位（横向+纵向，分析前必做）

**在走六步流程之前，先定位今天的叙事在时间轴和历史截面上的位置。**

这是为了确保每一天的分析不是孤立的，而是同一个故事的延续。

**横向（历史进程）** — 核心叙事在时间轴上怎么演化的：
1. 读最近2-3天的信源日报，回答：
   - 昨天/前天的核心叙事是什么？
   - 今天这个叙事是强化了、弱化了、还是转向了？
   - 有没有新的叙事在浮现？有没有旧的叙事在消退？
2. 标注当前状态：`一致性/分歧/转向/混沌`
   - 一致性 = 全市场在交易同一个逻辑
   - 分歧 = 数据面和价格面指向不同方向
   - 转向 = 核心叙事正在切换
   - 混沌 = 没有清晰主线

**纵向（事件叠加与交叉）** — 当前时间节点上，多事件的交汇效果：
1. 把今天同时发生的事件列出来（不分大小，先全抓）
2. 问：这些事件是互相强化、互相抵消、还是彼此独立？
3. 问：叠加后的合成方向是什么？（不要只看单个事件的影响）
4. 标注：`单线驱动 / 多线共振 / 多线对冲`

**输出到文章中**：这个定位结果写在"因果链展开"之前，可以是一小段（1-2句），不需要独立成章。例如：

> *"上周五非农之后市场在交易'加息重启'，但今天花旗逆势喊降息 + BTC崩跌 + 费城半导体-10%，叙事从一致性进入分歧期。数据面说加息，机构面说降息——市场在定价最坏情况，不是最可能情况。"*

这段 === "今天这个市场在发生什么"的顶层定性。

### 第1~6步：六步分析流程（同前）

在写任何内容之前，先走完六步分析流程（Dorian框架）：
1. **变量拆解** — 从六层数据中找到最关键的1-2个变量
2. **传导链** — 完整走完A→B→C→意味着什么
3. **历史对标** — 类似变量组合的历史 + 这次不一样的地方
4. **情景推演** — 三个情景：基准/乐观/悲观
5. **交易映射** — 进攻还是防守？方向倾向？
6. **失效条件** — 至少写2种具体的"什么情况说明我错了"

六步分析在脑子里走完，输出时按下面的五段结构写。

### 最终闭环：交易员检验（写完分析后必做）

写完上述所有内容后，问自己最后一句话：

> **"这份数据如果给一个交易员看，他接下来会做什么动作？"**

这不是要求写具体的买卖建议，而是强制把分析翻译成一个可观察、可执行的方向判断。几种合格的表达方式：

- *"明天大低开跑不掉，估计要跌两天测承接，不急着抄底。"*
- *"未来一周如果纳指再跌5%，可以分批接科技。"*
- *"现在不做方向赌注，但密切关注半导体ETF放量企稳的信号。"*

**不合格的：** *"需警惕风险，等待时机"*、*"谨慎乐观，防御优于进攻"*

判断标准：这句话能放在交易员晨会上说吗？如果能，过关。如果说完会被问"所以呢"——重写。

## 核心要求

### 1. 结构（按顺序，但不要写章节标题）

**A. 锚点事件开篇（1段）**
- 选当日最重要的1个具体事件作为开头，不是综述
- 事件必须有数据：时间、数字、对比
- 用一句话解释"为什么这个事件重要"

**B. 因果链展开（2-3段）**
- 从锚点事件出发，走完一条完整的因果链：A发生了什么 → 为什么A导致B → B传导到C → C意味着什么
- 每层转折要有数据或引用来支撑
- 不要只列数据，要解释"所以呢"

**C. 反直觉发现（1-2段）**
- 找出"表面上是X，实际上是Y"的矛盾点
- 例如：AI在跌但不是AI出问题，是利率在变
- 这是全文的洞察高峰，放在中间偏后

**D. 多源交叉验证（1段）**
- 同一个判断，有2-3个独立信源指向同一方向时，点出来
- 跨资产验证最有力：美债说X、商品说Y、股票说Z，三个方向一致
- 如果某数据源缺失，在这里标注"XX数据缺失，以下判断置信度降低"
- 如果shock_detector为空但市场明显在定价某事件，写"XX事件未被数据源捕获，但我观察到YY"

**E. 可操作的结论（1段）**
- 给出1-2个具体、可验证的判断：如果A发生则B，如果C发生则D
- 附带时间窗口：未来1个月、3个月、6个月
- 不要写"谨慎乐观/防御优于进攻"这类正确废话

### 2. 逻辑链思维

每一段回答三个问题中的一个：
- 为什么这个数据重要？（因果）
- 这个数据和什么有关？（关联）
- 这个数据能告诉我什么别人没看到的东西？（反直觉）

禁止：
- 平铺数据：不要"流动性0.113, VIX 25.21, M2 8.6%"
- 分块报告：不要"公众号部分...量化部分...宏观部分..."
- 正确废话：不要"需警惕风险，把握机会"

### 3. 语气和语言

- 用交易员的语言，不说"综上所述/由此可见"
- 句子短，一段不超过5行
- 可以打比方（猫笔刀风格），但比喻要贴切
- 数据引用要自然融入叙事，不是生硬插入
- 总字数控制在800-1200字

### 4. 数据源的融合顺序

优先顺序（按决策权重）：
1. 宏观锚定：流动性监控、US宏观、日本宏观 → 定方向
2. 消息面发酵：sentiment_shock → 定当前主要矛盾
3. 公众号观点：猫笔刀/表舅/卓哥/亨特/laoduo → 交叉验证 + 微观信号
4. 市场结构：US ETF/明星股/概念链动量 → 定资金流向
5. 基本面：数据层因子溢价 → 长期验证

## 输出示例风格（不是模板）

开头应该像：
"上周五非农就业17.2万，预期8万，超一倍。这个数字本身不重要，重要的是它引发的连锁反应——"

不是：
"根据最新数据，美国5月非农就业新增17.2万，超出预期。这一数据导致..."

## 每次必须覆盖的审视角度

1. 全球流动性在松还是紧？（流动性因子+日本套息+美元方向）
2. 市场当前的核心矛盾是什么？（sentiment_shock 中最集中的冲击类型）
3. 公众号们在关注什么同一件事？（多个信源同时指向的话题）
4. 资金在从哪去哪？（ETF/明星股/概念链的动量排序）
5. 有什么别人没提但我看到了的东西？（反直觉信号）

### 6. 超级IPO与流动性抽血（新增，非常重要）

**未来1-3个月有没有超大市值IPO/增发/解禁？** 这些事件直接抽取/释放市场流动性，影响程度往往超过宏观因子。

必须思考的问题链：
- 近期有什么超级IPO？（SpaceX 6/12上市募资750亿刀 / Anthropic秘密交表 / OpenAI年内上市）
- 这些IPO进入指数的时间表（SpaceX上市后最快15个交易日进纳指100 → 其他成分股被动卖出）
- 大厂再融资（谷歌850亿刀股权融资，巴菲特认购100亿）→ 市场抽血多少？
- 有没有"IPO排队效应"——多个大票集中上市对流动性的累积冲击
- 时间点有没有巧合？（比如SpaceX 6/12上市 vs 其他事件）

**写作指引**：这往往是"因果链中的关键转折"——"为什么资金从XX流出？因为下周SpaceX要抽走750亿"

### 7. 跨境资本流动的监管变化（新增，非常重要）

**最近有没有限制/开放资本跨境流动的政策变化？** 这类变化直接改变资金流向的"管道"结构。

必须审视的角度：
- 国内资金出海的通道有没有变化？（老虎/富途境外账户清退，6月12日起允许卖不允许买）
- QDII额度有没有变化？（持续收窄还是放宽）
- 人民币汇率波动对跨境资本的影响（离岸人民币升破6.76创3年新高）
- 港股通/债券通的额度或规则变化
- 时间点因果关系：为什么这个政策变化发生在某个特定时点？（如清退令生效日6/12 vs SpaceX上市日6/12）
- 外汇储备变化与外管局政策态度

**写作指引**：这类叙事最适合作为"反直觉发现"——"表面上是SpaceX上市抽血，但实际上中国提前一天把资金出海通道卡死了，这才是影响全球资本流动的关键变量"

### 8. 所有审视角度合成后的自检问题

写完因果链后，问自己：
- 这条链里有没有"超大IPO抽血"这个环节？如果没有 → 是否真的没有超级IPO？
- 这条链里有没有"跨境资本管制"这个环节？如果没有 → 是否真的没有政策变化？
- 如果两个同时存在，它们的时间点巧合说明了什么？

## 数据缺失时的推理（拆解）

如果核心数据源缺失，AI必须按以下逻辑推理并告知读者：

| 缺失的数据源 | 推理方向 | 在文章中如何表述 |
|:--|:--|:--|
| 公众号全空（API过期） | 缺失7个信源当日观点，无法交叉验证。结合宏观数据推测近期市场讨论焦点 | "今日公众号数据缺失（API可能已过期），但结合宏观数据判断，市场讨论大概率集中在XX/XX/XX" |
| shock_detector为空（API超时） | 可能遗漏了突发事件。检查其他数据中的异常值反向推断 | "消息面检测未返回数据，但VIX升至25+/韩股熔断/费城半导体-10%显示市场在定价系统风险，大概率有外部触发事件未被捕获" |
| 中国宏观快照为空 | 国债利率/汇率/商品数据丢失，无法判断国内政策方向 | 标注缺失，降低此段置信度 |
| US ETF/明星股全部为空 | 日度截面的资金流向排名丢失 | 用趋势延续推断（上周最强链/最弱链大概率延续），标注为推测 |
| 流动性数据为空 | 合成压力指数丢失 | 用各因子（VIX/BTC/DXY）分别描述，不做合成判断 |

## 检查清单（写完后自检）

- [ ] 过去信源日报读过了吗？知道之前写过什么判断吗？
- [ ] 横向（历史进程）定位做了吗？今天叙事是强化/弱化/转向？
- [ ] 纵向（事件叠加）分析做了吗？多事件是共振还是对冲？
- [ ] 今天核心矛盾和昨天比有没有变化？如有变化，有没有标注？
- [ ] 之前判断错了有没有认错和修正？
- [ ] 数据完整性自检是否做完了？
- [ ] 是否有数据源缺失？缺失的推理是否写进文章了？
- [ ] 六步流程在脑子里走完了吗？
- [ ] 失效条件至少有2条了吗？
- [ ] 审视角度6（超级IPO抽血）：近期有没有超大IPO/增发抽走流动性？
- [ ] 审视角度7（跨境资本管制）：近期有没有资本流动的管道变化？
- [ ] 如果角度6和7同时存在，它们的时点巧合分析了吗？
- [ ] 总字数800-1200？
- [ ] 没有"综上所述/由此可见"这类废话？
- [ ] 没有平铺数据（流动性0.113）？
- [ ] 开篇是一个具体事件，不是综述？
- [ ] 结论是可证伪的（如果A则B），不是"谨慎乐观"？
