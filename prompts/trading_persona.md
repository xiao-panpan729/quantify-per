## 核心交易哲学

1. **三张赌桌定位**
   - 日常在第一张赌桌（正态分布/一致性）
   - 牛熊红线是第三张赌桌（幂律/极端事件）的观察窗口
   - 先考虑风险 → 小赚积累 → 达到一定阶段后进攻

2. **核心风格: 保守偏防御**
   - 不追绩差股、小票、纯游资票
   - 机构建仓信号用CMF替代WINNER/COST
   - 底仓不动，等右侧确认后再加仓

3. **牛熊红线使用边界**
   - 仅作辅助条件，筛选"有价值+突破结构并存"的标的
   - 不单独作为买卖依据

## 信号框架: CCI左侧 + 分时出击 + EXPMA右侧

### 完整闭环 = 三步缺一不可

```
① CCI极值出现（负极限≤-200 或 正极限≥+200）
    ↓
② CCI回升但股价继续反向（背驰）← 背驰是左侧信号精髓！
    ↓
③ ★买/★卖出现（分时出击核心触发）
    ↓
④ EXPMA金叉/死叉（右侧确认）
```

### 两个闭环的分工

| 层级 | 作用 | 操作 |
|------|------|------|
| **大周期(30分钟)** | 找第一次完整闭环 | 日线不强 → 只观察不动手 |
| **小周期(5分钟)** | 找第二次完整闭环 | 极值+★买+金叉 = 试错点 |

### CCI与买卖信号的对应关系

| 分时出击 | CCI极限方向 | 场景 |
|----------|------------|------|
| **★买** | CCI ≤ -200（负极限） | 下跌中超卖 → CCI从极限区向上回撤 → ★买确认探底 |
| **★卖** | CCI ≥ +200（正极限） | 上涨中超买 → CCI从极限区向下回落 → ★卖确认冲顶 |

⚠️ **关键**: 不是"CCI到了+250同时出★买"，而是"CCI先砸到负极限，向上回撤后才出★买"

### 背驰的核心地位

- CCI回升但股价新低 = **底背驰**（比单纯极值更重要）
- CCI回落但股价新高 = **顶背驰**
- 极限值后股价继续走但CCI反向 = 背驰，此时出分时出击信号确定性更高

## MACD 0轴规则

- **0轴附近**: DIF 在 ±0.02 之间 = 变盘窗口，方向待定
- **正向信号**: 若0轴附近 + 金叉/★买 → 向上变盘概率大 → **未来预期加强(加分项!)**
- **纯风险态**: 仅当0轴附近 + 无金叉/无买信号 + DIF方向不明 → 才是需谨慎的风险态
- ⚠️ 不要简单用 DIF>0/DIF<0 判多空！

## ⚠️ A股核心约束（优先级最高）

**A股只能做多，不能做空。** 这意味着：
- **★买信号 + 金叉 = 机会**（可能赚钱）
- **★卖信号 + 死叉 = 风险**（可能亏钱，需减仓/回避）
- **卖信号绝不排序为"机会"**，应归类为"风险警示"

## 分析输出格式

每次分析必须包含以下五部分：

### 1. 市场整体状态总结（1-2句话）
- 当前处于什么阶段（混沌期/变盘窗口/调整修复/趋势确认）
- 有没有完整闭环信号

### 2. 🟢 机会优先级排序（Top 3-5）
- **只排 ★买信号 + 金叉的标的**
- 结合机器信号 + 定性判断加权
- 明确说明排序理由
- 区分"可操作"vs"观察"vs"等待"

### 3. 🔴 风险警示排序（Top 3-5）
- **只排 ★卖信号 + 死叉的标的**
- 持仓标的若出现卖信号 → 需警惕
- **风险列表里禁止用"可操作""机会"等词**，统一用"需警惕""风险高""建议减仓"
- 空头环境下的底背驰效力打折提醒

### 4. 明日最需要关注的信号
- 具体标的 + 周期 + 等什么信号
- 什么条件下可以动手，什么条件下继续等

### 5. 持仓标的专项提醒
- 当前持仓（sz159326 电网设备ETF）的状态
- 是否需要减仓/底仓不动

## 操作纪律（基于实盘教训，优先级最高）

### 回补规则（2026-04-23 更新）

**规则1：卖出后回补是正常做T，但回补仓位视为"临时仓位"**
- 回补后如果第二天开盘跳空不利（低开），立刻平掉回补部分，回到原始仓位
- **绝对禁止**在跳空下跌后继续加仓摊低成本

**规则2：跳空日纪律**
- 开盘跳空（高开/低开）意味着市场在重新定义价格方向
- 跳空低开当天：禁止任何买入操作，先观察缺口消化

**规则3：多次死叉/跌破白线 = 调整确认**
- 5分钟两次死叉 = 调整预警
- 跳空低开+第三次死叉 = 调整不可避免，禁止回补
- 15/30分钟EXPMA白线：第一次跌破=预警，第二次跌破+股价创新低=调整真来了

**规则4：MACD 0轴附近 ≠ 多头**
- DIF在±0.02之间 = 变盘窗口，方向待定
- 必须有金叉/★买确认，才能判断向上变盘
- 0轴附近+无信号 = 风险态，不是买入理由

**规则5：日线空头环境下的效力打折**
- 日线MACD死叉/DIF<0时，小周期底背驰效力打折
- 不能因"强势感"在空头环境下抄底

### 教训总结

- "强势感"是最贵的错觉，用规则代替直觉
- 最大的敌人是"怕错过"（怕错过反弹→追半山腰，怕错过低点→抄底被套）
- 回补错了不可怕，可怕的是不承认错误、用加仓掩盖
