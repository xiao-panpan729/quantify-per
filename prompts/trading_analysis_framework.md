# 战役级交易分析框架 v1.0
## 用于指导 AI 读取信号后做出判断的标准化模板

> 本文件定位：**分析逻辑框架**，不是角色人格。
> 角色人格见 trading_persona.md（谁在说），本文件决定"怎么分析、判断标准是什么"。
> 任何 AI（DeepSeek V4 / Claude / Qwen）接入信号数据后，按此框架输出分析。

---

## 一、分析总纲（第一性原理）

### 1.1 分析顺序（不可逆）

```
趋势判定 → 强度分级 → 位置评估 → 逐级信号扫描 → 操作决策 → 跟踪条件
```

**绝对禁止**：
- ❌ 跳过趋势判定直接看信号
- ❌ 罗列所有级别的信号（只说最重要的）
- ❌ 把卖信号排为"机会"

### 1.2 核心原则

| 原则 | 内容 |
|:--|:--|
| **趋势优先** | 定了趋势才能看信号。逆趋势操作需要多级共振 |
| **只看最强** | 定了趋势后只看最强的买入机会，不罗列所有级别 |
| **具体位置** | 说"黄白线之间 / 前一波50%回调位"，不说"等回调到EXPMA附近" |
| **级别精确** | 给具体的闭环数量要求，不模糊 |
| **论证简洁** | 分析2-3句、判断1句、结论1句、跟踪1句 |

## 二、第一层：趋势定位

### 2.1 趋势方向判定

| 维度 | 多头 | 震荡 | 空头 |
|:--|:--|:--|:--|
| EXPMA12 vs 50 | e12 > e50 | 粘合/缠绕 | e12 < e50 |
| MACD | DIF>0 + DEA>0 | 0轴附近 | DIF<0 + DEA<0 |
| 金叉/死叉 | 金叉状态 | 无/刚切换 | 死叉状态 |
| 趋势评分(0-14) | 8-10 顺势窗口 | 3-7 中性 | 0-2 筑底 |
| 评分进阶 | 11+ 虚高警示: 看起来强但易反转 | | |
| 验证数据 | 8分+0.73% 9分+0.88% 10分+1.12% 最强 | | 11分+0.27%(虚高) 12分-1.95% 13分-2.87%(反转) |

**一句话论证**：EXPMA金叉+MACD双多=多头确认。MACD在0轴附近无金叉=变盘窗口，方向待定。

### 两轴决策框架

评分系统分为两个独立轴，同时使用：

| 轴 | 名称 | 来源 | 回答的问题 | 用法 |
|:--|:--|:--|:--|:--|
| **纵轴** | 操作级别 (ABCD) | macd_score(0-4) | 信号在哪个周期可信？ | 周期筛选：A=min5, B=min5, C=min15, D=min30+ |
| **横轴** | 环境建议 (zone_advice) | total_score(0-14) | 现在该不该做？ | 环境判断：sweet_spot=可做, fragile_high=保守 |

两轴独立，不互相替代。例如：A级 + fragile_high → "可以做小周期但环境虚高，严格止损"。

### 2.2 操作级别（ABCD — 周期筛选）

| 等级 | 条件 | 最小操作级别 | 一句话判断 |
|:--|:--|:--:|:--|
| **A最强** | 价格在EXPMA12白线上方 | 5分钟一信号 | 强势趋势，顺势为主 |
| **B次强** | 白线下方、黄线上方 | 5分钟★买+2次金叉 | 中性偏强，需确认 |
| **C偏弱** | 黄线下但MACD>0 | 15分钟★买+2次金叉 | 弱势反弹，等大级别 |
| **D弱势** | MACD<0或死叉 | 不参与 | 不操作 |

### 2.3 位置评估

| 位置 | 特征 | 应对 |
|:--|:--|:--|
| **低位区** | 价格接近/低于EXPMA50黄线 | 关注★买机会，准备入场 |
| **中位区** | 黄白线之间 | 等回调不破再入场 |
| **高位加速区** | 远超EXPMA50(>+20%) | 持有减仓，不追多 |
| **极端偏离** | 偏离黄线>+40% | 只减不加，等大幅回调 |

**偏离度评估公式**：
```
偏离白线% = (close - expma12) / expma12 × 100
偏离黄线% = (close - expma50) / expma50 × 100
```

## 三、第二层：信号扫描

### 3.1 完整闭环链条

```
CCI极值(≤-200/≥+200)
    ↓
背驰（看面积非高度，底背驰/顶背驰）
    ↓
★买 / ★卖 信号出现
    ↓
EXPMA金叉 / 死叉 确认
```

**论证说明**：四个环节缺一不可。背驰比极值更重要（面积比高度更可信）。

### 3.2 逐级信号检查（从上到下）

```
日线 → 60分钟 → 30分钟 → 15分钟 → 5分钟
```

每一级检查：
1. ★买/★卖 信号几根K线内出现过？
2. EXPMA金叉/死叉 状态？
3. CCI是否极端？是否有背驰？
4. 信号质量评分（buy_level / sell_level）？

**论证说明**：上一级信号决定下一级的操作基调。日线多→可做多；日线空→等大级别底部。

### 3.3 闭环摘要规则

- 不列出所有级别
- 找最重要的2-3个级别（日线必看+1个主操作级别+次级别参考）
- 描述★买/★卖模式：交替出现 / 成比例出现 / 单边密集
- 弱势/有瑕疵的闭环：说明为什么不能用，降级看下一级
- 跟踪条件精确：如"多等2次5分钟闭环买才有机会"

## 四、第三层：操作决策

### 4.1 顺势操作条件（做多/加仓/做T）

**适用场景**：A/B级趋势，当前方向与趋势一致。

| 条件 | 说明 |
|:--|:--|
| ① 次级别(5分钟)出现★买+金叉 | 核心触发 |
| ② MA5上穿MA10 | 试错信号（可提前半步） |
| ③ 前后两个闭环买的低点抬高 | 趋势健康验证 |
| ④ 如果之前减仓了，这里接回 | 做T闭环 |

**一句话论证**：A级趋势+5分钟★买+金叉+低点抬高=顺势做T/加仓。不用等多级共振，方向就是最大保证。

### 4.2 逆向操作条件（减仓/卖出）

**适用场景**：需要多级共振才能出手，防止被趋势甩下车。

**三级共振条件**（全部满足才出手）：

```
第一级：操作级别（15分钟）出现★卖
        +
第二级：同级别共振（15分钟★卖 + 30分钟★卖）
        +
第三级：次级别确认（5分钟出现≥2组闭环卖信号=★卖+死叉）
```

**退出检查**：次级别（5分钟）出现★买+金叉=新的闭环买→检查从卖到买是否有利润空间→有则逆向操作成功。

**论证说明**：A级趋势中，单一级别★卖大概率是假摔，必须多级共振才能确认转向意图。

### 4.3 持有不动条件

当以下情况同时出现：
- ✅ 趋势方向未变（A/B级）
- ❌ 没有新的顺势闭环买信号（不加仓）
- ❌ 逆向多级共振条件未满足（不减仓）
- ❌ 偏离度极端但无信号（不单凭偏离度操作）

**论证说明**：持有不是消极等待，而是在等待两个信号之一出现——要么新顺势买点加仓，要么多级卖共振减仓。

### 4.4 战役级转向信号（大级别）

| 信号 | 含义 | 操作 |
|:--|:--|:--|
| 日线★卖 + 死叉 | 趋势可能结束 | 大幅减仓/清仓 |
| HHT判定"趋势结束/剧烈衰减" | 结构破坏 | 减仓至轻仓 |
| 低点没有抬高反而降低（缠论） | 趋势不健康 | 卖出成功验证 |
| 突破关键支撑位（前低） | 破位确认 | 全部离场 |

**论证说明**：大级别转向必须有日线级信号加持，小级别★卖只是预警不是转向。

---

### 4.5 C类偏弱趋势分析规则

**定义**：价格在EXPMA50黄线下方，但MACD>0。

**核心逻辑**：弱势反弹格局，操作门槛显著高于A/B级。目标是**做反弹，不是趋势多**。

| 维度 | 规则 |
|:--|:--|
| **最小操作级别** | 15分钟★买 + 2次金叉 |
| **共振要求** | 15分钟+30分钟共振，5分钟≥2组闭环买 |
| **"必须强更强"** | 前一个日线闭环是★卖时，目标只是做反弹。不能只因为"有信号"就认为可操作 |
| **低点验证** | 每个金叉的低点必须依次抬高。跌破前低=趋势不顺畅，不做多 |
| **60分钟有效性** | ★买+金叉后价格应运行在黄白线之间。跌破黄线+跌回涨幅一大半=很弱，没有买的信号 |
| **降级链条** | 日线有瑕疵→60分钟弱→30/15分钟需要共振→5分钟需要双闭环才可信 |
| **EXPMA加分项** | 如果前面有金叉+回撤黄线附近，属于偏多积极信号，可加分 |

**一句话论证**：C类趋势的本质是"有瑕疵的反弹"。不要因为下级有信号就认为可以操作，必须层层降级确认到5分钟双闭环，才具备最低可信度。

### 4.6 D类弱势趋势分析规则

**定义**：MACD<0或死叉状态。

**核心逻辑**：不参与，等大级别底部。不存在"抄底试错"的概念。

| 维度 | 规则 |
|:--|:--|
| **可操作性** | 不参与 |
| **跟踪价值** | ⭐ 不跟踪 |
| **最小重新参与级别** | 日线级完整闭环（★买+金叉+MACD上穿0轴） |
| **禁止行为** | ❌ 不因小级别底背驰而入场 ❌ 不因"强势感"在空头环境抄底 |

**一句话论证**：MACD<0代表大级别空头压制，任何小级别信号效力都会打折。等日线MACD金叉+站回白线，那才是重新评估的起点。

### 4.7 按强度分级操作速查表

| 强度 | 位置 | 操作基调 | 最小买入条件 | 减仓/卖出条件 |
|:--|:--|:--|:--|:--|
| **A级** | 白线上方 | 顺势做多/做T | 1个5分钟★买+金叉 | 15+30分钟★卖共振+5分钟≥2闭环卖 |
| **B级** | 黄白线之间 | 偏多，需确认 | 2个5分钟★买+金叉 | 同上 |
| **C级** | 黄线下，MACD>0 | 做反弹，不是趋势多 | 15分★买+2次金叉+共振 | 反弹结束信号出现即走 |
| **D级** | MACD<0 | 不参与 | 等日线完整闭环 | — |

## 五、特殊规则

### 5.1 偏离度规则

**偏离EXPMA黄线特别远时**：修正ABC D级别的判断——A级趋势但偏离>+40%，实际风险已接近B/C级。操作上以减仓锁定利润为主。

### 5.2 "必须强更强"规则

适用：震荡走势中、日线前一个闭环是★卖、但死叉后立刻金叉的标的。

判断标准：
- 每个金叉的低点必须依次抬高 = 趋势健康
- 跌破前低 = 趋势不顺畅，不做多
- 60分钟★买+金叉后价格跌破黄线 + 跌回涨幅一大半 = 很弱，没有买的信号

### 5.3 大盘系数修正

| 大盘状态 | 系数 | 影响 |
|:--|:--:|:--|
| 上涨/偏多 | 1.0-1.2 | 顺势操作可积极 |
| 震荡/中性 | 0.8-1.0 | 正常操作 |
| 偏空/下跌 | 0.6-0.8 | 降低仓位、提高共振门槛 |

## 六、分析输出格式模板

### 模板结构

```
## 分析标的：{代码} {名称}

### 1. 当前定位（1-2句）
方向：{上涨/震荡/下跌} | 强度：{A/B/C/D}级 | 位置：{低位/中位/高位加速/极端偏离}
评分：{X}/14 [{zone_label}] | 大盘系数：{X}
→ 一句话总结趋势定位

**zone_advice 使用注意**：
- sweet_spot(8-10分) = 真实强势区，顺势做多窗口
- fragile_high(11+分) = 虚高警示：MACD+MA完美但闭环薄弱，**来路决定去向**
  - 从低位涨上来的高分可以延续→做多但严格止损
  - 高位二次冲高的高分是陷阱→不追高，等分回落
- fragile_low(0-2分) = 筑底观察，不抄底
- neutral(3-7分) = 中性等待

### 2. 关键信号（1-2句）
日线：buy_level=X sell_level=X | {★买密集/★卖密集/均衡} | {顶背驰/底背驰/无}
{主操作级别}：{★买/★卖}出现 | {金叉/死叉} | buy_level=X
→ 一句话说明最重要的信号

### 3. 操作决策（1句）
操作：{做多/加仓/持有/持有减仓/观望卖出/全部离场}
依据：{引用上面判断的依据，用→串联}

### 4. 跟踪条件（1句）
等：{具体条件，不出现"看分钟信号"字样}
    ✅ "等回调EXPMA12后30分钟找★买"
    ✅ "等5分钟★买+金叉+低点抬高"
    ❌ "等回调到EXPMA12附近再看分钟信号"
```

### 完整示例

```
## 分析标的：sh513310 中韩半导体ETF

### 1. 当前定位
方向：上涨 | 强度：A级 | 位置：高位加速区，偏离黄线+48%
评分：10/14 | 大盘系数：0.8(偏空)
→ A级趋势但已进入极端偏离区，大盘背景不支持共振。

### 2. 关键信号
日线：buy=5.0 sell=4.3 | ★买密集(6次) | 逆向崩退(熵↓)
15分钟：★卖出现(10:30) | 30分钟无★卖共振
→ 15分钟单级★卖，不足以触发减仓。

### 3. 操作决策
操作：持有，观望
依据：A级趋势完整→不全面卖出；30分钟无★卖共振→不减仓；无新★买+金叉→不加仓。

### 4. 跟踪条件
等：回调EXPMA12(5.158)附近→30分钟★买+金叉→做T接回。
如5分钟连续出现★卖+死叉×2组→减1/3防守。
```

## 七、常见错误与禁止清单

| 错误类型 | 错误示例 | 正确做法 |
|:--|:--|:--|
| 跳趋势看信号 | "15分钟有★买，可操作" | 先定趋势，再看信号是否匹配 |
| 罗列所有级别 | "日线XX/60分XX/30分XX/15分XX/5分XX" | 只写最重要的2-3级 |
| 卖信号当机会 | "★卖+金叉，关注买入" | ★卖是风险警示，归类到风险栏 |
| 位置表述模糊 | "等回调到EXPMA12附近" | "等黄白线之间"或"等前一波50%" |
| 条件不精确 | "等分钟信号" | "等1个5分钟闭环"或"等2次5分钟★买" |
| 仅凭偏离度操作 | "偏离+48%了，必须卖" | 偏离度是风控提示，不是操作信号，等多级共振 |
| 顶背驰=卖出 | "连续顶背驰，卖出" | 顶背驰是预警，需要等★卖+共振才动手 |
